## ⚠️ 存在的问题

### 问题 1：风格统一性难保证 ⭐⭐⭐⭐⭐

**现象**：
- AI 每次生成的风格可能不同
- 像素密度不一致（有时粗有时细）
- 颜色饱和度不稳定
- 线条粗细不统一

**原因**：
- MJ 的随机性
- Nano Banana 的不稳定性
- 缺少统一的调色板
- 缺少后期统一处理

**影响**：
- 主角、伙伴、敌人风格不统一
- 看起来像拼凑的
- 不够专业

---

### 问题 2：AI 理解提示词不准确 ⭐⭐⭐⭐

**现象**：
- "FLOATING" 可能生成成站立
- "DIFFERENT" 可能生成 4 帧相同
- "64×64" 数字被忽略
- "透明背景" 可能有白边

**原因**：
- AI 的理解能力有限
- 提示词再详细也有概率失败
- 需要多次重试

**影响**：
- 需要多次生成才能得到满意结果
- 浪费时间和成本

---

### 问题 3：缺少质量控制流程 ⭐⭐⭐⭐

**现象**：
- 没有明确的"合格标准"
- 没有"不合格如何修正"的指南
- 没有后期处理流程

**原因**：
- 文档只有"验收"，没有"修正"
- 依赖 AI 一次生成成功

**影响**：
- 遇到问题不知道怎么办
- 可能接受不合格的资源

---

### 问题 4：缺少风格指南 ⭐⭐⭐⭐⭐

**现象**：
- 没有统一的调色板
- 没有统一的像素密度标准
- 没有统一的光源方向
- 没有统一的线条粗细

**原因**：
- 文档只有"生成流程"，没有"风格规范"

**影响**：
- 每个角色风格不同
- 整体不协调

## 🎯 改进建议

### 改进 1：建立美术风格指南 ⭐⭐⭐⭐⭐

**目标**：统一所有美术资源的风格

**内容**：

#### 1.1 调色板（8-16 色）

```
修仙肉鸽 - 标准调色板

【主色调】
- 浅蓝：#5BA3D0（道袍、水系）
- 深蓝：#2E5C7A（阴影）
- 白色：#F0F0F0（高光、云雾）
- 米色：#D4C4A8（皮肤）

【五行色】
- 火红：#E74C3C（火系、赤渊）
- 水蓝：#3498DB（水系、玄霄）
- 木绿：#27AE60（木系、青璃）
- 金黄：#F39C12（金系、庚金）
- 土褐：#8B6F47（土系、碧落）

【辅助色】
- 黑色：#2C3E50（头发、轮廓）
- 灰色：#7F8C8D（阴影）
- 紫色：#9B59B6（特效、墨羽）
```

**使用方法**：
1. 生成后用 Aseprite 统一调色
2. 在提示词中指定颜色（如 "vibrant light blue #5BA3D0"）

---

#### 1.2 像素密度标准

```
【推荐】粗像素风格（64×64 效果）
- 角色宽度：32-48 像素
- 角色高度：64-80 像素
- 每个像素块：3-5 像素宽
- 参考：8-bit NES/Game Boy

【备选】细像素风格（128×128 效果）
- 角色宽度：60-70 像素
- 角色高度：100-110 像素
- 每个像素块：1-2 像素宽
- 参考：16-bit SNES/Genesis
```

**建议**：统一使用粗像素风格（更有特色，性能更好）

---

#### 1.3 光源方向

```
统一光源：左上方 45°

【高光位置】
- 头顶左侧
- 肩膀左侧
- 道袍左侧褶皱

【阴影位置】
- 右下方
- 身体右侧
- 道袍右侧褶皱
```

---

#### 1.4 线条粗细

```
【轮廓线】2 像素（粗像素）或 1 像素（细像素）
【内部线】1 像素
【细节线】1 像素或无
```

### 改进 2：添加质量控制流程 ⭐⭐⭐⭐⭐

**目标**：确保每个资源都合格

**流程**：

```
生成 → 检查 → 修正 → 再检查 → 接入
```

#### 2.1 检查清单

**必须项（不合格必须重做）**：
- [ ] 2×2 网格，4 帧
- [ ] PNG 透明背景，无白边
- [ ] 像素风格，无抗锯齿
- [ ] 每帧有明显差异
- [ ] 角色正面，居中

**推荐项（不合格可修正）**：
- [ ] 颜色符合调色板
- [ ] 像素密度统一
- [ ] 光源方向统一
- [ ] 线条粗细统一

---

#### 2.2 常见问题修正指南

**问题 A：4 帧相同或差异太小**

**修正方法**：
1. 重新生成（调整提示词，强调 "DIFFERENT"）
2. 手动修改（用 Aseprite 调整姿态）
3. 程序化生成（用 procedural_anim.py）

**问题 B：背景不透明或有白边**

**修正方法**：
1. 用 Aseprite 或 Photoshop 去除白边
2. 用魔棒工具选择背景 → 删除
3. 用 Python 脚本批量处理：

```python
from PIL import Image
import numpy as np

def remove_white_bg(img_path, output_path, threshold=240):
    img = Image.open(img_path).convert("RGBA")
    data = np.array(img)
    
    # 白色像素变透明
    white = (data[:,:,:3] > threshold).all(axis=2)
    data[white, 3] = 0
    
    Image.fromarray(data).save(output_path)
```

**问题 C：像素密度不统一**

**修正方法**：
1. 用 Aseprite 缩放到统一尺寸（64×64 或 128×128）
2. 用 "Nearest Neighbor" 算法（保持像素风）
3. 批量处理脚本：

```python
from PIL import Image

def resize_pixel_art(img_path, output_path, size=(64, 64)):
    img = Image.open(img_path)
    img = img.resize(size, Image.NEAREST)  # 保持像素风
    img.save(output_path)
```

**问题 D：颜色不符合调色板**

**修正方法**：
1. 用 Aseprite 的 "Replace Color" 功能
2. 用 Photoshop 的 "Color Lookup Table"
3. 用 Python 脚本映射到最近的调色板颜色

### 改进 3：优化提示词 ⭐⭐⭐⭐

**目标**：提高 AI 理解准确率

#### 3.1 添加参考图 URL（MJ）

**当前**：
```
A young Chinese male cultivator...
```

**优化**：
```
https://[参考图URL] A young Chinese male cultivator...
```

**效果**：风格更统一

---

#### 3.2 强化关键词（Nano Banana）

**当前**：
```
FLOATING in ALL 4 frames
```

**优化**：
```
CRITICAL: FLOATING in ALL 4 frames—NEVER sitting, NEVER standing on ground
REPEAT: The character is FLOATING, NOT sitting, NOT standing
```

**效果**：AI 更容易理解

---

#### 3.3 添加负面提示词

**当前**：
```
No glow, no aura, no particles
```

**优化**：
```
NEGATIVE PROMPT: sitting, standing on ground, identical frames, white background, anti-aliasing, smooth gradients, high resolution, glow, aura, particles, effects
```

**效果**：避免不想要的元素

---

### 改进 4：添加后期处理流程 ⭐⭐⭐⭐

**目标**：统一风格，提升质量

**流程**：

```
AI 生成 → 去白边 → 统一尺寸 → 统一调色 → 添加细节 → 导出
```

**工具**：
- Aseprite（推荐，专业像素画工具）
- Photoshop（备选）
- Python 脚本（批量处理）

**步骤**：

1. **去白边**（5 分钟）
   - 魔棒选择白色 → 删除
   - 或用脚本批量处理

2. **统一尺寸**（5 分钟）
   - 缩放到 64×64 或 128×128
   - 用 Nearest Neighbor 算法

3. **统一调色**（10 分钟）
   - 用 Aseprite 的 "Replace Color"
   - 映射到标准调色板

4. **添加细节**（可选，10 分钟）
   - 修正轮廓线
   - 添加高光/阴影
   - 调整对比度

5. **导出**（1 分钟）
   - PNG，透明背景
   - 保存到 assets 文件夹

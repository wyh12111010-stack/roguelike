# 数值与平衡规范（Balanced 版）

> 目标：把“可玩”升级为“可控、可复盘、可迭代”的数值系统。  
> 难度定位：`balanced`（理解机制后可通关，容错有限但不苛刻）。

---

## 二、Balanced 难度目标

### 2.1 目标体验

- 新玩家 2~4 局进入可持续推进状态。
- 熟悉机制后，5 局内应有稳定通关可能。
- 失败可归因，不出现“看不懂为何暴毙”。

### 2.2 量化指标（核心 KPI）

| 指标 | 目标区间 | 说明 |
|------|----------|------|
| 全流程通关率 | 25%~40% | 以熟悉基础机制玩家为样本 |
| 平均单局时长 | 28~40 分钟 | 低于 25 分钟偏简单，高于 45 分钟偏拖沓 |
| 首次 Boss 击败率（妖王） | 55%~70% | 教学 Boss 需给学习空间 |
| 终局 Boss 击败率（秽源） | 25%~40% | 维持挑战感 |
| 平均死亡归因可解释率 | >=85% | 能归因为走位/资源/机制失误 |

## 三、战斗预算模型

### 3.1 TTK 预算（玩家击杀敌人）

| 目标对象 | 目标 TTK | 体感目标 |
|----------|----------|----------|
| 普通近战/远程 | 0.8~2.0s | 不卡手，但不能秒屏 |
| 精英单位 | 2.5~5.0s | 明显更硬，要求走位 |
| 单 Boss 阶段 | 35~90s | 有学习与适应窗口 |

### 3.2 生存预算（敌人击杀玩家）

| 场景 | 玩家可承受失误（命中） | 说明 |
|------|------------------------|------|
| 章节 1 普通关 | 8~10 次 | 新手教学容错 |
| 章节 2 普通关 | 6~8 次 | 压力上升 |
| 章节 3 普通关 | 4~6 次 | 资源管理阶段 |
| Boss 高危技 | 4~5 次 | 可怕但可读 |

### 3.3 Boss 时长预算

| Boss | 目标时长 |
|------|----------|
| 妖王 | 70~100s |
| 剑魔 | 90~130s |
| 丹魔 | 90~130s |
| 秽源 | 120~180s |

## 四、章节难度曲线（建议）

### 4.1 曲线原则

- 章节递进 > 单关递进：同章节小关波动小，避免“某一关断崖”。
- 先增压迫类型，再增裸数值：优先引入新敌人行为，不先堆伤害。
- Boss 是章节考试，不是纯肉墙。

### 4.2 三章节结构

| 章节 | 主要压力源 | 数值趋势 | 机制趋势 |
|------|------------|----------|----------|
| 试炼 | 追击与基础弹道 | 低-中 | 教玩家读前摇与换位 |
| 秘境 | 冲刺/AOE 组合 | 中 | 开始要求技能/药资源管理 |
| 凶地 | 追踪/召唤/多线程 | 中-高 | 强化目标优先级与控场 |

### 4.3 失败归因目标（整局）

| 归因类型 | 目标占比 |
|----------|----------|
| 站位失误 | 50%~60% |
| 资源管理失误（技能/CD/药） | 25%~35% |
| 识别机制错误 | 10%~20% |
| 随机波动 | <=10% |

## 五、属性与伤害口径（与代码一致）

### 5.1 第一层乘算（保留）

| 情况 | 倍率 |
|------|------|
| 相合（灵根=法宝） | 1.20 |
| 相异反应 | 1.10 |
| 无加成 | 1.00 |

### 5.2 第二层反应（保留当前值）

继续使用：斩木、破土、堵水、灭火、熔金 的当前参数，不在本轮做硬改。

### 5.3 调参顺序（固定）

1. 敌人行为与密度  
2. Boss 技能节奏  
3. 敌人伤害  
4. 玩家恢复/容错  
5. 元素倍率

这样可避免“只调数字掩盖机制问题”。

## 六、局外成长策略（Balanced 推荐）

- 短期：维持 `base_health_bonus` / `base_mana_bonus` 默认关闭，仅作为后续柔性补偿阀。
- 中期：若通关率低于 20%，开放小幅成长（每档 +10，上限 +30）。
- 长期：不靠无限膨胀数值解决难度，以机制学习和构筑选择为主。

## 七、验收与回归模板

### 7.1 每轮改动必须记录

- 改动内容（行为/数值/掉落）
- 目标指标（例如妖王击败率 +10%）
- 实测结果（至少 3 局）
- 结论（保留/回滚/继续调）

### 7.2 回归最小清单

- 普通关：章节 0/1/2 各打 1 关
- Boss：妖王 + 最终 Boss 各 1 局
- 结果：时长、死亡原因、药耗、资源盈余

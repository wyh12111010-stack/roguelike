# 成就系统

> 栖霞处成就功能已实现，成就条目已接入游戏逻辑。

---

## 一、框架说明

- **入口**：中央广场（ROOM_CENTER）与栖霞对话 → 成就面板
- **存储**：`meta.achievements` 为已解锁成就 ID 集合
- **判定**：游戏逻辑中调用 `unlock_achievement(id)` 解锁

---

## 二、已实现成就条目

| 分类 | ID | 名称 | 描述 |
|------|-----|------|------|
| 战斗 | first_victory | 初凯旋 | 首次凯旋 |
| 战斗 | kill_10/30/100 | 十斩/三十斩/百斩 | 累计击杀 10/30/100 敌人 |
| 战斗 | level_5/10 | 过关五/过关十 | 累计过关 5/10 次 |
| 战斗 | death_1/3 | 身陨/三陨 | 首次/累计 3 次身陨 |
| 伙伴 | heal_partner | 初愈 | 首次治疗伙伴 |
| 伙伴 | heal_all_partners | 全愈 | 治疗全部 5 位伙伴 |
| 成长 | unlock_linggen | 灵根初开 | 解锁任意灵根 |
| 成长 | unlock_fabao | 法宝初得 | 解锁任意法宝 |
| 探索 | enter_event | 奇遇 | 首次进入奇遇节点 |

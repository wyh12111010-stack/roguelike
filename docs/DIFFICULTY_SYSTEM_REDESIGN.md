# 难度递增与重复刷价值设计（重做版）

> 问题 1：天劫系统与世界观矛盾（主角是世界意识，不受天劫）
> 问题 2：缺少重复刷的价值（过了就过了，没有反复刷的理由）

---

## 一、核心问题分析

### 1.1 世界观约束
```
主角身份：世界意识的最后代表，拥有轮回之力
村子：世界意识保留的最后锚点
秽源：外域恶意的具象化

矛盾：
  主角 = 世界意识 → 不应该受「天劫」
  天劫 = 修真者的考验 → 与主角身份不符
  
正确方向：
  应该是「秽源越来越强」
  而不是「主角受天劫」
```

### 1.2 Hades/星座战士的重复刷价值
```
Hades：
  - 每个热度等级都有独特奖励
  - 特定热度解锁特定对话/剧情
  - 特定热度解锁特定物品
  - 武器每个热度都有独特挑战
  → 玩家会反复刷同一热度

星座战士：
  - 每个觉醒等级都有独特掉落
  - 特定觉醒解锁特定法术
  - 特定觉醒解锁特定角色
  - 每个觉醒都有独特成就
  → 玩家会反复刷同一觉醒

我们的问题：
  - 通关后只是难度+1
  - 没有特定等级的独特奖励
  - 没有反复刷的理由
```

## 二、方案 A：秽源侵蚀等级 + 独特奖励池

### 2.1 核心概念：秽源侵蚀度

#### 世界观融合
```
设定：
  每次击败秽源，它会吸收更多外域力量
  下次出现时，侵蚀度更高
  侵蚀度越高，秽源的力量越强
  但同时，也会掉落更珍贵的「净化结晶」

侵蚀度 0：初次对抗（基础难度）
侵蚀度 1：秽源吸收外域力量，变强
侵蚀度 2：秽源进一步侵蚀，出现新形态
...
侵蚀度 10：秽源接近完全体，终极挑战
```

#### 关键设计：每个侵蚀度都值得反复刷
```
侵蚀度 1：
  - 解锁「净化结晶·壹」掉落池
  - 只有侵蚀度 1 才会掉落特定饰品
  - 特定成就只能在侵蚀度 1 完成
  
侵蚀度 2：
  - 解锁「净化结晶·贰」掉落池
  - 只有侵蚀度 2 才会掉落特定法宝
  - 特定伙伴只能在侵蚀度 2 解锁
  
...

关键：
  - 不是「侵蚀度越高越好」
  - 而是「每个侵蚀度都有独特价值」
  - 玩家会反复刷特定侵蚀度
```

### 2.2 独特奖励设计

#### 侵蚀度 0（基础）
```
掉落池：
  - 基础饰品
  - 基础法宝
  - 基础道韵

解锁：
  - 无特殊解锁

成就：
  - 首次通关
  - 基础成就
```

#### 侵蚀度 1
```
掉落池：
  - 「侵蚀之心」（饰品，只在侵蚀度 1 掉落）
  - 「净化之刃」（法宝，只在侵蚀度 1 掉落）
  - 道韵 +20%

解锁：
  - 新伙伴：赤渊（只能在侵蚀度 1 首次遇到）
  - 新灵根：金/土

成就：
  - 「初次侵蚀」：首次通关侵蚀度 1
  - 「侵蚀猎手·壹」：侵蚀度 1 通关 10 次
  - 「无伤侵蚀·壹」：侵蚀度 1 无伤通关

特殊：
  - 赤渊的特殊对话（只在侵蚀度 1 触发）
```

#### 侵蚀度 2
```
掉落池：
  - 「深度侵蚀核心」（饰品，只在侵蚀度 2 掉落）
  - 「反侵蚀法杖」（法宝，只在侵蚀度 2 掉落）
  - 道韵 +40%

解锁：
  - 新伙伴：碧落（只能在侵蚀度 2 首次遇到）
  - 新灵根：无灵根
  - 新事件类型

成就：
  - 「深度侵蚀」：首次通关侵蚀度 2
  - 「侵蚀猎手·贰」：侵蚀度 2 通关 10 次
  - 「速通侵蚀·贰」：侵蚀度 2 在 25 分钟内通关

特殊：
  - 碧落的特殊对话
  - 新 Boss 技能
```

#### 侵蚀度 3-9
```
每个侵蚀度都有：
  - 独特掉落（特定饰品/法宝）
  - 独特解锁（伙伴/灵根/事件）
  - 独特成就（通关/无伤/速通/特殊条件）
  - 独特对话（伙伴/NPC）
  - 道韵递增（+60%, +80%, +100%...）
```

#### 侵蚀度 10（终极）
```
掉落池：
  - 「世界意志碎片」（最强饰品）
  - 「终焉法宝」（最强法宝）
  - 道韵 +200%

解锁：
  - 真结局
  - 隐藏伙伴
  - 隐藏模式

成就：
  - 「终极侵蚀」：首次通关侵蚀度 10
  - 「侵蚀终结者」：侵蚀度 10 通关 10 次
  - 「完美净化」：侵蚀度 10 无伤通关
```

### 2.3 侵蚀度选择机制

#### 在村子中选择
```
┌─────────────────────────────┐
│   栖霞村 - 出发准备           │
│                             │
│   当前侵蚀度：2              │
│   最高已通关：5              │
│                             │
│   选择侵蚀度：               │
│   ○ 侵蚀度 0（基础）         │
│   ○ 侵蚀度 1（+20% 道韵）    │
│   ● 侵蚀度 2（+40% 道韵）    │
│   ○ 侵蚀度 3（+60% 道韵）    │
│   ○ 侵蚀度 4（+80% 道韵）    │
│   ○ 侵蚀度 5（+100% 道韵）   │
│   ○ 侵蚀度 6（未解锁）       │
│                             │
│   侵蚀度 2 独特奖励：        │
│   - 深度侵蚀核心（饰品）     │
│   - 反侵蚀法杖（法宝）       │
│   - 碧落的特殊对话           │
│                             │
│   [确认] [取消]              │
└─────────────────────────────┘
```

#### 关键设计
```
1. 玩家可以自由选择侵蚀度（已解锁的）
2. 不是「必须打侵蚀度 N 才能打 N+1」
3. 而是「打过侵蚀度 N 后，N+1 解锁」
4. 玩家可以反复刷任意侵蚀度

例如：
  - 打过侵蚀度 5 后，可以回去刷侵蚀度 2
  - 因为侵蚀度 2 有独特掉落
  - 或者想要侵蚀度 2 的特定成就
```

## 三、方案 B：秽源形态 + 分支路线

### 3.1 核心概念：秽源的多种形态

#### 世界观融合
```
设定：
  秽源不是单一存在，而是外域恶意的集合
  每次被击败后，会以不同形态重生
  不同形态有不同能力和掉落

形态列表：
  - 狂暴形态：高伤害，掉落攻击类饰品
  - 坚韧形态：高生命，掉落防御类饰品
  - 迅捷形态：高速度，掉落速度类饰品
  - 侵蚀形态：召唤多，掉落特殊饰品
  - 混沌形态：随机能力，掉落稀有物品
```

#### 关键设计：每种形态都值得反复刷
```
狂暴形态：
  - 敌人伤害 +50%
  - 掉落「狂暴之心」（伤害+25% 饰品）
  - 掉落「狂暴法宝」系列
  - 道韵 +30%
  
坚韧形态：
  - 敌人生命 +80%
  - 掉落「坚韧之盾」（减伤+15% 饰品）
  - 掉落「坚韧法宝」系列
  - 道韵 +30%
  
迅捷形态：
  - 敌人速度 +60%
  - 掉落「迅捷之翼」（移速+20% 饰品）
  - 掉落「迅捷法宝」系列
  - 道韵 +30%

关键：
  - 玩家根据想要的掉落选择形态
  - 每种形态都有独特价值
  - 不是「越难越好」
```

### 3.2 形态选择 + 难度叠加

#### 基础形态（侵蚀度 0）
```
5 种基础形态：
  - 狂暴、坚韧、迅捷、侵蚀、混沌
  - 每种都有独特掉落
  - 难度相当
```

#### 强化形态（侵蚀度 1+）
```
每个基础形态都可以叠加侵蚀度：
  - 狂暴形态 + 侵蚀度 1
  - 狂暴形态 + 侵蚀度 2
  - ...
  
例如：
  狂暴形态 + 侵蚀度 3：
    - 敌人伤害 +50%（狂暴）+ 45%（侵蚀度 3）= +95%
    - 掉落：狂暴系列 + 侵蚀度 3 独特物品
    - 道韵：+30%（狂暴）+ 60%（侵蚀度 3）= +90%
```

#### 选择界面
```
┌─────────────────────────────┐
│   选择秽源形态               │
│                             │
│   基础形态：                 │
│   ● 狂暴（高伤害）           │
│   ○ 坚韧（高生命）           │
│   ○ 迅捷（高速度）           │
│   ○ 侵蚀（召唤多）           │
│   ○ 混沌（随机）             │
│                             │
│   侵蚀度：                   │
│   ○ 0（基础）               │
│   ● 1（+15% 难度，+20% 道韵）│
│   ○ 2（+30% 难度，+40% 道韵）│
│   ...                       │
│                             │
│   当前选择：                 │
│   狂暴形态 + 侵蚀度 1        │
│   敌人伤害 +65%              │
│   道韵 +50%                  │
│   独特掉落：狂暴之心、...    │
│                             │
│   [确认] [取消]              │
└─────────────────────────────┘
```

## 四、方案 C：污染层级 + 探索深度

### 4.1 核心概念：污染区域的深度

#### 世界观融合
```
设定：
  栖霞村外的世界被污染
  污染有深度：表层、中层、深层、核心
  越深入，污染越强，但奖励越好
  
层级：
  表层（0-3 关）：轻度污染
  中层（4-7 关）：中度污染
  深层（8-11 关）：重度污染
  核心（12-15 关）：极度污染
```

#### 关键设计：每个层级都值得反复刷
```
表层探索：
  - 难度低
  - 掉落基础物品
  - 适合新手/测试 Build
  - 快速刷道韵
  
中层探索：
  - 难度中
  - 掉落进阶物品
  - 平衡难度和收益
  
深层探索：
  - 难度高
  - 掉落稀有物品
  - 高风险高收益
  
核心探索：
  - 难度极高
  - 掉落传说物品
  - 终极挑战

关键：
  - 玩家可以选择探索深度
  - 不是「必须打完 15 关」
  - 可以只打表层（快速刷）
  - 可以只打中层（平衡）
  - 可以挑战深层（极限）
```

```

例如：
  - 只有 10 分钟？刷表层
  - 想要中层的特定饰品？刷中层
  - 想要终极挑战？刷核心
```

## 六、推荐方案

### 推荐：方案 A（侵蚀度）+ 方案 C（深度）混合

#### 核心设计
```
1. 基础：污染深度（4 选项）
   - 表层（3 关）：快速刷
   - 中层（7 关）：平衡
   - 深层（11 关）：挑战
   - 核心（15 关）：终极

2. 进阶：侵蚀度（10 等级）
   - 侵蚀度 0-10
   - 每个侵蚀度独特奖励
   - 可叠加在任意深度上

3. 组合：
   - 表层 + 侵蚀度 0：新手入门
   - 表层 + 侵蚀度 5：快速刷高级物品
   - 中层 + 侵蚀度 3：平衡刷
   - 核心 + 侵蚀度 10：终极挑战
```

#### 选择界面
```
┌─────────────────────────────┐
│   出发准备                   │
│                             │
│   探索深度：                 │
│   ● 中层探索（7 关）         │
│      基础道韵：+50%          │
│      时长：~20 分钟          │
│                             │
│   侵蚀度：                   │
│   ● 侵蚀度 3                 │
│      额外道韵：+60%          │
│      敌人强度：+45%          │
│                             │
│   总道韵加成：+110%          │
│   独特掉落：                 │
│   - 侵蚀度 3 专属饰品        │
│   - 中层专属法宝             │
│                             │
│   [确认] [取消]              │
└─────────────────────────────┘
```

#### 重复刷价值
```
1. 想要侵蚀度 3 的特定饰品？
   → 反复刷「中层 + 侵蚀度 3」

2. 想要快速刷道韵？
   → 反复刷「表层 + 侵蚀度 5」

3. 想要终极挑战？
   → 挑战「核心 + 侵蚀度 10」

4. 想要测试新 Build？
   → 选择「表层 + 侵蚀度 0」

关键：
  - 每个组合都有价值
  - 玩家根据目标选择
  - 不是「越高越好」
```

## 七、实施步骤

### 步骤 1：基础框架（立即）
```
1. 添加探索深度选择（4 选项）
2. 添加侵蚀度变量（0-10）
3. 根据深度和侵蚀度调整难度
4. 根据深度和侵蚀度调整道韵
```

### 步骤 2：独特奖励（1 周）
```
1. 设计每个侵蚀度的独特掉落
2. 设计每个深度的独特掉落
3. 实现掉落池系统
4. 测试平衡性
```

### 步骤 3：渐进式解锁（2 周）
```
1. 初始只开放表层 + 侵蚀度 0
2. 通关后解锁中层
3. 通关后解锁侵蚀度 1
4. 逐步解锁更多选项
```

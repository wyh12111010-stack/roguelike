# 中文字体显示问题修复报告

## 问题：左上角UI文字显示为方框

### 原因分析
文字显示为方框（□□□）说明：
1. 字体加载失败
2. 或者字体不支持中文字符

### 解决方案

**方案A：使用系统中文字体（推荐）**
修改 `config.py` 中的 `get_font` 函数，使用小写字体名称

**方案B：使用TTF字体文件**
下载中文TTF字体文件放到项目中

**方案C：临时方案**
将UI文字改为英文

---

## 问题：视角不跟随人物移动

### 原因
战斗区域是固定的，玩家在区域内移动，没有实现摄像机跟随

### 解决方案
这可能是设计选择：
- Tiny Rogues 风格：固定视角，单屏战斗
- 不需要摄像机跟随

如果需要摄像机跟随，需要：
1. 添加 Camera 类
2. 所有绘制坐标减去摄像机偏移
3. 摄像机跟随玩家中心

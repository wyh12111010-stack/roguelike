### 问题4：粒子效果不消失 ✅

**问题描述**：
- 粒子有时不消失
- 可能是列表清理问题

**修复方案**：
- 优化 `particles.py` 的update方法
- 改用列表推导式清理死亡粒子
- 避免在迭代时修改列表

**修改文件**：
- `particles.py` - 修改update方法

**修改内容**：
```python
# 原代码（可能有问题）
for p in self.particles[:]:
    p.update(dt)
    if p.dead:
        self.particles.remove(p)

# 新代码（更安全）
for p in self.particles:
    p.update(dt)
self.particles = [p for p in self.particles if not p.dead]
```

# 难度系统设计：侵蚀度 × 现有关卡结构

> 基于现有的 3 大关结构（试炼/秘境/凶地），对标 Hades 和死亡细胞

---

## 📊 现有关卡结构

```
试炼（大关 1）：关卡 0-4 → 休息/商店 → 妖王 Boss
  ↓
秘境（大关 2）：关卡 5-9 → 休息/商店 → 剑魔 Boss
  ↓
凶地（大关 3）：关卡 10-14 → 休息/商店 → 丹魔 Boss → 休整点 → 秽源 Boss

总计：15 个战斗关 + 3 个段末 Boss + 1 个终极 Boss
时长：约 28-40 分钟
```

---

## 🎯 难度系统设计：侵蚀度（类似死亡细胞 BC）

### 核心概念

**世界观融合**：
```
设定：
  每次击败秽源，它会吸收更多外域力量
  下次出现时，侵蚀度更高
  侵蚀度影响整个世界的污染程度
  
侵蚀度 0：初次对抗（基础难度）
侵蚀度 1：秽源吸收外域力量，敌人变强
侵蚀度 2：秽源进一步侵蚀，出现新敌人
...
侵蚀度 10：秽源接近完全体，终极挑战
```

**关键设计**：
- ✅ 线性解锁（类似死亡细胞）
- ✅ 每个侵蚀度都有独特内容
- ✅ 可以回刷任意已解锁的侵蚀度
- ✅ 不改变关卡数量（仍然是 3 大关）

---

## 📋 侵蚀度等级设计

### 侵蚀度 0（基础）

**难度**：
- 敌人属性：基础（100%）
- 敌人行为：基础
- Boss 技能：基础

**奖励**：
- 道韵：基础
- 掉落：基础饰品/法宝

**解锁**：
- 默认开放

**成就**：
- 「初次净化」：首次通关

---

### 侵蚀度 1

**难度**：
- 敌人属性：+15%（生命/伤害）
- 新敌人：侵蚀小怪（在试炼大关出现）
- Boss 技能：妖王增加 1 个新技能

**奖励**：
- 道韵：+20%
- 独特掉落：「侵蚀之心」（饰品，只在侵蚀度 1 掉落）
- 独特掉落：「净化之刃」（法宝，只在侵蚀度 1 掉落）

**解锁**：
- 通关侵蚀度 0 后解锁
- 新伙伴：赤渊（只能在侵蚀度 1 首次遇到）

**成就**：
- 「初次侵蚀」：首次通关侵蚀度 1
- 「侵蚀猎手·壹」：侵蚀度 1 通关 10 次
- 「无伤侵蚀·壹」：侵蚀度 1 无伤通关

**特殊**：
- 赤渊的特殊对话（只在侵蚀度 1 触发）
- 新事件：「侵蚀之源」

---

### 侵蚀度 2

**难度**：
- 敌人属性：+30%
- 新敌人：深度侵蚀怪（在秘境大关出现）
- Boss 技能：剑魔增加 1 个新技能

**奖励**：
- 道韵：+40%
- 独特掉落：「深度侵蚀核心」（饰品）
- 独特掉落：「反侵蚀法杖」（法宝）

**解锁**：
- 通关侵蚀度 1 后解锁
- 新伙伴：碧落（只能在侵蚀度 2 首次遇到）

**成就**：
- 「深度侵蚀」：首次通关侵蚀度 2
- 「侵蚀猎手·贰」：侵蚀度 2 通关 10 次
- 「速通侵蚀·贰」：侵蚀度 2 在 25 分钟内通关

---

### 侵蚀度 3-9（递增）

**每个侵蚀度都有**：
- 敌人属性递增：+45%, +60%, +75%, +90%, +105%, +120%, +135%
- 新敌人/新行为
- Boss 新技能
- 独特掉落（特定饰品/法宝）
- 独特解锁（伙伴/灵根/事件）
- 独特成就
- 道韵递增：+60%, +80%, +100%, +120%, +140%, +160%, +180%

---

### 侵蚀度 10（终极）

**难度**：
- 敌人属性：+150%
- 所有敌人都是强化版
- Boss 全新技能组
- 秽源 Boss 第三阶段

**奖励**：
- 道韵：+200%
- 独特掉落：「世界意志碎片」（最强饰品）
- 独特掉落：「终焉法宝」（最强法宝）

**解锁**：
- 通关侵蚀度 9 后解锁
- 真结局
- 隐藏伙伴
- 隐藏模式

**成就**：
- 「终极侵蚀」：首次通关侵蚀度 10
- 「侵蚀终结者」：侵蚀度 10 通关 10 次
- 「完美净化」：侵蚀度 10 无伤通关

## 🎮 玩家体验流程

### 第一次游玩

```
1. 村子中选择「侵蚀度 0」（默认）
2. 进入试炼大关（关卡 0-4）
3. 通过休息/商店
4. 击败妖王 Boss
5. 进入秘境大关（关卡 5-9）
6. 通过休息/商店
7. 击败剑魔 Boss
8. 进入凶地大关（关卡 10-14）
9. 通过休息/商店
10. 击败丹魔 Boss
11. 通过休整点
12. 击败秽源 Boss
13. 通关！解锁侵蚀度 1
```

### 第二次游玩

```
1. 村子中选择「侵蚀度 1」
2. 发现新敌人「侵蚀小怪」

4. 通关后获得「侵蚀之心」饰品
5. 遇到新伙伴「赤渊」
6. 解锁侵蚀度 2
```

### 第 N 次游玩

```
场景 1：想要侵蚀度 3 的特定饰品
  → 选择侵蚀度 3
  → 反复刷直到掉落

场景 2：想要快速刷道韵
  → 选择侵蚀度 5（+100% 道韵）
  → 快速通关

场景 3：想要挑战自己
  → 选择侵蚀度 10
  → 终极挑战
```

## 🔄 重复刷价值

### 为什么要反复刷同一侵蚀度？

**1. 独特掉落**
```
想要「侵蚀之心」饰品？
  → 只有侵蚀度 1 掉落
  → 反复刷侵蚀度 1

想要「深度侵蚀核心」饰品？
  → 只有侵蚀度 2 掉落
  → 反复刷侵蚀度 2
```

**2. 独特成就**
```
想要「无伤侵蚀·壹」成就？
  → 必须在侵蚀度 1 无伤通关
  → 反复挑战侵蚀度 1

想要「侵蚀猎手·贰」成就？
  → 必须侵蚀度 2 通关 10 次
  → 反复刷侵蚀度 2
```

**3. 独特对话/剧情**
```
想看赤渊的特殊对话？
  → 只在侵蚀度 1 触发
  → 刷侵蚀度 1

想看碧落的特殊对话？
  → 只在侵蚀度 2 触发
  → 刷侵蚀度 2
```

**4. 道韵效率**
```
想要快速刷道韵？
  → 侵蚀度 5（+100% 道韵）
  → 难度适中，效率高

想要极限刷道韵？
  → 侵蚀度 10（+200% 道韵）
  → 难度极高，但奖励最高
```

## 📈 与 Hades/死亡细胞对比

| 维度 | Hades（热度） | 死亡细胞（BC） | 我们（侵蚀度） |
|------|--------------|---------------|---------------|
| **线性解锁** | ❌ 自由选择 | ✅ 线性 | ✅ 线性 |
| **可回刷** | ✅ 任意热度 | ❌ 通常不回刷 | ✅ 任意侵蚀度 |
| **独特内容** | ✅ 每武器×热度 | ✅ 每 BC | ✅ 每侵蚀度 |
| **新手友好** | ⭐⭐ 选项多 | ⭐⭐⭐⭐ 线性清晰 | ⭐⭐⭐⭐ 线性清晰 |
| **重复刷价值** | ⭐⭐⭐⭐ 很高 | ⭐⭐⭐ 较高 | ⭐⭐⭐⭐⭐ 极高 |
| **世界观契合** | ⭐⭐ 契约 | ⭐⭐⭐ 细胞 | ⭐⭐⭐⭐⭐ 侵蚀完美 |

## ✅ 优势总结

### 1. 保持现有结构
- ✅ 不改变 3 大关结构
- ✅ 不改变关卡数量
- ✅ 不改变时长（28-40 分钟）

### 2. 线性解锁（类似死亡细胞）
- ✅ 新手友好，不会迷失
- ✅ 明确的进度感
- ✅ 逐步解锁新内容

### 3. 可回刷（类似 Hades）
- ✅ 每个侵蚀度都有独特价值
- ✅ 玩家根据目标选择
- ✅ 不是「越高越好」

### 4. 世界观契合
- ✅ 侵蚀度符合秽源设定
- ✅ 不是「天劫」（主角不受天劫）
- ✅ 符合修仙世界观

## 🚀 实施步骤

### 步骤 1：基础框架（1 天）
```
1. 在 meta.py 添加 erosion_level 变量
2. 在村子界面添加侵蚀度选择
3. 根据侵蚀度调整敌人属性
4. 根据侵蚀度调整道韵奖励
```

### 步骤 2：独特掉落（2 天）
```
1. 设计每个侵蚀度的独特饰品/法宝
2. 实现掉落池系统
3. 根据侵蚀度过滤掉落
4. 测试掉落概率
```

### 步骤 3：渐进式解锁（1 天）
```
1. 初始只开放侵蚀度 0
2. 通关后解锁下一侵蚀度
3. 记录最高通关侵蚀度
4. 可以回刷任意已解锁侵蚀度
```

### 步骤 4：成就与对话（可选）
```
1. 为每个侵蚀度添加成就
2. 为伙伴添加侵蚀度相关对话
3. 添加新事件
```

## 🎯 总结

**核心设计**：
- **侵蚀度 0-10**：线性解锁，类似死亡细胞 BC
- **可回刷**：任意已解锁侵蚀度，类似 Hades 热度
- **独特内容**：每个侵蚀度都有独特掉落/成就/对话
- **保持结构**：不改变现有的 3 大关结构

**重复刷价值**：
- ✅ 独特掉落（只在特定侵蚀度掉落）
- ✅ 独特成就（只在特定侵蚀度解锁）
- ✅ 独特对话（只在特定侵蚀度触发）
- ✅ 道韵效率（高侵蚀度高道韵）

**实施时间**：3-4 天

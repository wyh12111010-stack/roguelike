### 1.3 难度系统实现 ⏳

**目标**：实现秽源侵蚀度系统，提供重复刷价值

**任务**：
- [ ] 实现侵蚀度选择界面
- [ ] 实现侵蚀度对敌人的影响
- [ ] 实现侵蚀度独特奖励池
- [ ] 实现探索深度选择（表层/中层/深层/核心）

**设计方案**：
```
侵蚀度 0-10：
  - 侵蚀度 0：新手模式，敌人 -20% 属性
  - 侵蚀度 1-3：正常难度
  - 侵蚀度 4-6：困难难度，敌人 +30% 属性
  - 侵蚀度 7-9：噩梦难度，敌人 +60% 属性
  - 侵蚀度 10：地狱难度，敌人 +100% 属性

探索深度：
  - 表层（3 关）：快速刷，10 分钟
  - 中层（7 关）：平衡，20 分钟
  - 深层（11 关）：挑战，30 分钟
  - 核心（15 关）：终极，40 分钟

独特奖励：
  - 每个侵蚀度有独特饰品掉落
  - 每个深度有独特物品
  - 玩家根据目标选择组合
```

**产出**：
- `erosion_system.py` 完整实现
- 村子界面添加侵蚀度/深度选择
- `levels.py` 支持动态关卡数量

**验收标准**：
- 可以自由选择侵蚀度和深度
- 难度曲线合理
- 每个组合都有重复刷价值

**预计时间**：3-4 天

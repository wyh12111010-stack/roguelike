## 🎮 核心设计亮点

### 1. 秽源共鸣系统

```
设计理念：
  - 对标 Hades 热度系统
  - 简化选项（18 个 vs Hades 32 个）
  - 同类互斥（降低复杂度）
  - 每个共鸣等级有专属掉落

重复刷价值：
  - 想要特定饰品？刷对应共鸣等级
  - 想要快速刷道韵？选高共鸣强度
  - 想要测试 Build？选低共鸣强度

世界观契合：
  - 秽源侵蚀（符合设定）
  - 共鸣强度（而非契约）
  - 净化奖励（道韵倍率）
```

### 2. 特色饰品系统

```
设计原则：
  - 横向差异，而非纵向递增
  - 每个饰品都有代价
  - 鼓励组合，而非单核

示例：
  狂暴碎片：伤害+12%，生命-10
  狂暴之核：暴击率+15%，攻速-8%
  狂暴之心：连击>10时伤害+30%

→ 三个饰品各有特点
→ 没有明显强弱
→ 适合不同流派
```

### 3. 27 关卡结构

```
设计理念：
  - 3 大关 × 9 小关 = 27 关
  - 单局 45-60 分钟
  - Build 有足够时间成型

对比：
  - Hades：约 48 个房间，30-40 分钟
  - 死亡细胞：约 25-30 个房间，25-35 分钟
  - 我们：27 关，45-60 分钟 ✓
```

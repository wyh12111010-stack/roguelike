# 村子地图设计

> 紧凑布局，房间相邻，直接走过去，准备工作顺手

---

## 一、设计原则

1. **直接走过去**：房间相邻，无需连接，玩家直接移动即可
2. **紧凑顺手**：灵根、法宝、传送门放近，选完即可外出
3. **村民分散**：村民在各自负责的房间内出现，每人有 NPC 功能
4. **局外成长**：后续可合理将局外成长功能添加到村民身上

---

## 二、房间规划（参考 PARTNER_VILLAGER_INITIAL.md）

| 房间 | 功能 | 对应村民 |
|------|------|----------|
| **灵根殿** | 选灵根、解锁灵根 | 玄真 |
| **炼器坊** | 选法宝、解锁法宝 | 铸心 |
| **中央** | 出生点 | 栖霞（引导） |
| **传送门厅** | 外出、演示入口 | - |
| **玄霄** | 治疗→伙伴 | 玄霄（雷宗掌门） |
| **青璃** | 治疗→伙伴 | 青璃（青丘狐族长老） |
| **赤渊** | 治疗→伙伴 | 赤渊（散修剑尊） |
| **碧落** | 治疗→伙伴、base_health/mana | 碧落（丹道宗师） |
| **墨羽** | 治疗→伙伴 | 墨羽（影宗宗主） |

5 位可治疗伙伴的房间**平均分布在村庄四角及右中**：左上角（玄霄）、左下角（墨羽）、右上角（青璃）、右中（赤渊）、右下角（碧落），画面和谐不扎堆。

---

## 三、地图尺寸

- **地图大小**：1200 × 700（扩展以容纳 5 伙伴平均分布）
- **相机**：跟随玩家
- **房间**：相邻摆放，走几步即可切换

## 五、实现要点

- `village.py`：地图尺寸、房间 rect、相机偏移
- `game.py`：村子玩家移动时应用地图边界，绘制时应用相机偏移
- 村民：在对应房间内添加 NPC 实体，可对话、治疗、局外成长等

# 美术资源问题诊断与解决方案

> 诊断时间：2025-02-27
> 问题：单独看状态图看不出动态效果，预览时人物中心每帧不一样

---

## 🔍 问题分析

### 问题 1：单独看状态图看不出动态效果

**这是正常的！**

**原因**：
- 状态图是 2×2 网格的精灵表（sprite sheet）
- 4 帧静态图排列在一起
- 需要用工具按顺序播放才能看到动画效果

**类比**：
- 就像电影胶片，单独看每一格看不出动作
- 需要连续播放才能看到动画

**结论**：✅ 这不是问题，是正常现象

---

### 问题 2：预览时人物中心每帧不一样

**这是 AI 生成的常见问题！**

**原因**：
- AI 生成每帧时，角色在画布上的位置不固定
- 导致每帧的"质心"（重心）位置不同
- 如果直接播放，角色会"跳来跳去"

**示意图**：
```
帧1: 角色偏左上    帧2: 角色偏右下
┌─────────┐      ┌─────────┐
│  👤      │      │      👤  │
│         │      │         │
└─────────┘      └─────────┘

播放效果：角色会"跳动"
```

**结论**：⚠️ 这是 AI 生成的问题，需要修正

---

## 🎯 解决方案

### 方案 A：用预览工具的对齐功能（推荐）⭐⭐⭐⭐⭐

**你的预览工具应该已经有这个功能了！**

**操作**：
1. 运行预览工具：
```bash
python tools/sprite_preview.py
```

2. 加载状态图

3. **按 A 键切换对齐模式**
   - 默认：居中对齐（按画布中心）
   - 按 A：质心对齐（按角色重心）

4. 观察效果：
   - 质心对齐后，角色应该不会"跳动"
   - 动画应该流畅

**如果按 A 后还是跳动**：
→ 说明 AI 生成的帧差异太大
→ 需要用方案 B 或 C

---

### 方案 B：用脚本自动对齐（推荐）⭐⭐⭐⭐⭐

**原理**：
- 计算每帧的质心（重心）
- 把所有帧的质心对齐到同一位置
- 输出对齐后的精灵表

**我可以帮你写这个脚本！**

**脚本功能**：
```python
# 输入：原始精灵表（2×2 网格）
# 输出：对齐后的精灵表（2×2 网格）
# 
# 步骤：
# 1. 分割成 4 帧
# 2. 计算每帧的质心
# 3. 把所有帧的质心对齐
# 4. 合并成新的精灵表
```

**使用方法**：
```bash
python tools/align_sprite_sheet.py nanobanana_in/主角待机图.png -o nanobanana_in/主角待机图_对齐.png
```

**优点**：
- ✅ 自动化，批量处理
- ✅ 不需要手动调整
- ✅ 效果稳定

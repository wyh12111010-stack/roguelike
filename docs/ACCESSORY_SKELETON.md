# 饰品系统设计（方案 D）

> 已拍板：6 槽 + 可强化 + 属性派系 + 功能派系

---

## 一、槽位（已拍板）

- **默认 6 槽**：开局就有 6 个饰品槽
- 满槽时获得新饰品 → 必须选一个替换（或放弃）
- **道韵扩槽**：可扩到 7、8、9 槽，作为后期目标
- 详见 `FABAO_ACCESSORY_DESIGN.md`

---

## 二、派系设计（方案 D）

### 主维度：属性派系（与灵根、法宝统一）

| 派系 | 主题 | 饰品倾向 |
|------|------|----------|
| **火** | 爆发、灼烧 | 伤害、暴击、灼烧相关 |
| **水** | 续航、流动 | 回血、灵力恢复、减伤 |
| **雷** | 迅捷、连锁 | 移速、攻速、连锁伤害 |
| **冰** | 控制、稳固 | 减速、护甲、无敌 |
| **毒** | 持续、侵蚀 | 持续伤害、击杀回血 |
| **无/金** | 纯粹、均衡 | 基础数值、通用加成 |

### 副维度：功能定位

- **生存**：生命、减伤、护甲、回血
- **输出**：伤害、暴击、攻速、元素加成
- **机动**：移速、闪避、冷却
- **辅助**：灵力恢复、伙伴充能、灵石加成

### 派系加成

- **同属性多件**：2 件同系 → 额外加成；3 件 → 更强加成
- **不同属性**：可设计元素反应类联动（如火+水→蒸发增伤）

---

## 三、品质 + 强化（已拍板）

- **品质**：普通 / 稀有 / 史诗，局内掉落时随机，影响基础强度
- **可强化**：每件饰品都可多次强化（过关、灵石、事件等）
- 强化时可选：派系加成 or 通用数值
- 品质 + 可升级结合：既有「开出稀有」惊喜，又有「普通强化上去」成长感

- 后续：减伤、灵石加成、伙伴充能、派系联动等

## 六、获取途径（仅局内）

- **村子不选饰品**：饰品只在局内存在，开局槽位空
- **局外成长**：可解锁「随机携带 1 饰品开局」（可选，可限制品质）

| 途径 | 品质池 | 说明 |
|------|--------|------|
| 商店 | 普通 + 稀有 | 花灵石，刷新机制 |
| 过关奖励 | 普通 | 清敌后二选一 |
| Boss 掉落 | 稀有 + 史诗 | 击败 Boss 后选一 |
| 事件 | 随机 | 奇遇/抉择给 |

---

## 七、替换流程

1. 槽位未满 → 直接装备
2. 槽位已满 → 选一个替换，或放弃
3. 被换下的饰品本局消失（除非再次掉落）

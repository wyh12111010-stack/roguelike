## 🎮 对标游戏 UI 分析

### Tiny Rogues（主要参考）
**特点**：
- ✅ 极简主义：只显示必要信息
- ✅ 边缘布局：所有 UI 在屏幕边缘
- ✅ 中央留白：战斗区域完全清爽
- ✅ 图标化：大量使用图标代替文字
- ✅ 半透明：UI 不遮挡游戏内容

**布局**：
```
┌─────────────────────────────┐
```

## 🎨 我们的 UI 设计方案

### 设计原则
1. **极简主义**：只显示必要信息
2. **边缘布局**：中央留给战斗
3. **图标化**：减少文字，增加图标
4. **修仙风格**：古风配色 + 符文装饰
5. **信息层级**：重要信息大且亮

## 🎨 视觉风格设计

### 配色方案（修仙古风）

**主色调**：
- 背景：深邃夜空 (15, 18, 25)
- 面板：半透明黑 (20, 25, 35, 230)
- 边框：金铜色 (180, 150, 90)

**强调色**：
- 生命：朱红 (220, 60, 60)
- 灵力：青灵 (60, 140, 200)
- 金币：仙金 (218, 165, 32)
- 道韵：碧玉 (64, 224, 208)

**文字颜色**：
- 标题：金色 (255, 215, 0)
- 正文：米黄 (240, 230, 210)
- 说明：浅灰 (180, 170, 150)
- 暗淡：灰色 (120, 110, 100)

---

### 装饰元素

**1. 边框装饰**
```
╔═══════════════════╗
║                   ║
╚═══════════════════╝
```
- 使用古风纹路
- 四角有装饰（云纹、回纹）
- 边框 2-3 像素宽

**2. 图标边框**
```
┌─────┐
│ 图标 │ ← 金色边框
└─────┘
```
- 当前选中：金色发光边框
- 未选中：深色边框
- 空槽：虚线边框

**3. 分隔线**
```
━━━━━━━━━━━━━━━━━━━
```
- 顶部和底部之间
- 使用渐变色（中间亮，两端暗）

## 🎮 交互设计

### 法宝切换

**按键**：Tab 或 Q
**效果**：
1. 两个法宝图标交换位置（0.2 秒动画）
2. 高亮边框切换
3. 播放切换音效
4. 显示切换提示（中央，0.5 秒）

```
     切换前              切换后
┌─────────┐  ┌─────────┐
│ 赤炎剑  │  │ 玄水符  │
│ (高亮)  │  │ (高亮)  │
└─────────┘  └─────────┘
┌─────────┐  ┌─────────┐
│ 玄水符  │  │ 赤炎剑  │
│         │  │         │
└─────────┘  └─────────┘
```

---

### 饰品悬停

**触发**：鼠标悬停在饰品图标上
**效果**：
1. 图标放大 10%
2. 显示详细信息（浮动窗口）
3. 显示饰品等级和效果

```
┌────────────────────┐
│ 暴烈玉 Lv.2        │
│ ─────────────────  │
│ 伤害 +16%          │
│ (基础 +8% × 2级)   │
└────────────────────┘
      ↑
   [图标]
```

## 🚀 总结

### 核心改进
1. **极简布局**：顶部状态 + 底部装备，中央留给战斗
2. **图标化**：大量使用图标，减少文字
3. **修仙风格**：古风配色 + 符文装饰
4. **信息层级**：重要信息大且亮，次要信息小且暗

### 实现建议
- 先实现 P0（核心 UI）
- 再实现 P1（交互优化）
- 最后实现 P2（视觉打磨）

### 预期效果
- UI 质量提升 200%+
- 符合 Tiny Rogues 设计哲学
- 强化修仙风格
- 提升用户体验

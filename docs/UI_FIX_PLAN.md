# UI修复方案

## 问题1：UI重复显示 ✅

**原因**：
- `CombatSystem._draw_ui` 在左上角绘制信息
- `ui_manager.draw` 在下方绘制HUD
- 两者重复显示了相同信息

**解决方案**：
- 移除 `CombatSystem._draw_ui` 中的重复信息
- 只保留左上角的关键信息（道韵、灵石、药水、关卡）
- HUD显示玩家状态（血量、灵力、法宝）

---

## 问题2：画布太小，左上角UI挡住战斗 ✅

**解决方案**：
- 增大画布尺寸：1280×720 → 1600×900
- 调整战斗区域位置，避开UI
- 优化UI布局，更紧凑

---

## 问题3：人物和敌人立绘未加载 ✅

**原因**：
- 代码中使用占位符矩形绘制
- 没有加载精灵图

**解决方案**：
- 修改 `player.py` 加载玩家精灵
- 修改 `enemy.py` 加载敌人精灵
- 使用 `config_assets.py` 获取路径

---

## 问题4：粒子效果不消失 ✅

**原因**：
- 粒子生命周期管理有bug
- 可能是粒子列表清理不及时

**解决方案**：
- 检查 `particle.py` 的生命周期逻辑
- 确保过期粒子被移除

---

## 问题5：灵力和技能CD机制不合理 ✅

**当前问题**：
- 灵力满了但技能CD还没好
- 玩家体验不佳

**解决方案**：
- 方案A：技能只消耗灵力，无CD
- 方案B：技能有CD，但CD很短（1-2秒），主要靠灵力限制
- 方案C：灵力满自动减少技能CD

**推荐**：方案B - 短CD + 灵力消耗

# 节点与路线设计

> 第一项待构想。本文档记录节点类型、路线结构、命名与出现规则。

---

## 参考：其他游戏的大关/小关数量

| 游戏 | 大关（Boss 段/世界/章节） | 小关（每大关内房间/节点数） | 一局总节点 |
|------|---------------------------|-----------------------------|------------|
| **死亡细胞** | 3～4 段（每段 1 Boss） | 每段 2～4 个普通关 | 牢房→2～3 关→Boss→2～3 关→Boss→…→王座 |
| **星界战士** | 4 个世界 | 每世界约 10+ 房间 | 4 世界 × 10+ 房 + 4 星座 Boss + 蛇夫座 |
| **哈迪斯** | 4 个 biome | 每 biome 约 10+ 房间 | 4 段 × 10+ 房 + 4 Boss |
| **小骨** | 5 个章节 | 每章若干小关 + 1 Boss | 5 章 × 若干关 + 5 Boss |

**常见规律**：大关 3～5 个，每大关内小关 2～10+ 不等；哈迪斯/星界战士小关多，死亡细胞/小骨相对紧凑。

## 参考方向：死亡细胞 + 星界战士

### 死亡细胞（Dead Cells）特点

| 方面 | 特点 |
|------|------|
| **拓扑结构** | 关卡间连接**固定**，牢房→大道/下水道→壁垒/藏骨堂…，玩家选出口 |
| **关卡内** | 房间布局**程序生成**，每关内自由探索、找出口 |
| **分支选择** | 每关结束选下一关出口，不同路线难度/奖励/符文需求不同 |
| **路线规划** | 玩家可提前规划「走哪条路」，有策略感 |

### 星界战士（Astral Ascent）特点

| 方面 | 特点 |
|------|------|
| **节点选择** | 过关后**直接选下一个房间类型**，选完即进，无关卡内探索 |
| **房间类型** | 战斗/休息/商店/选奖励/伙伴/小 Boss 等，**类型丰富** |
| **难度星级** | 选路前可见房间难度，玩家按当前强度选 |
| **世界主题** | 每大关有独立主题（地球/火/水/雷），Boss 在终点 |

### 融合倾向（修仙肉鸽）

- **拓扑**：采用死亡细胞式**固定路线树**，每层可选分支明确，玩家可规划路线
- **节点**：采用星界战士式**丰富节点类型**，过关后选战斗/商店/休息/事件/精英等
- **选路信息**：选路时显示**节点类型 + 难度/奖励倾向**，便于决策
- **关卡内**：当前为俯视角房间战斗，可保持「进房→清敌→选下一节点」，不做死亡细胞式关卡内自由探索（实现成本高）

## 参考：星界战士（Astral Ascent）路线设计

### 整体结构
- **4 个世界**：地球、红荒（火）、珊瑚群岛（水）、绯红高地（雷），主题与敌人各异
- **每世界终点**：Boss 房，击败星座守护者后进入下一世界
- **12 星座 Boss**：每世界随机 1 个，最终面对蛇夫座

### 房间类型（非常丰富）
| 类型 | 说明 |
|------|------|
| **探索房** | 清敌、开宝箱、拿货币，有锁区需钥匙 |
| **战斗房** | 波次战斗，拿钥匙，有时有星座挑战（限时/无伤等） |
| **日蚀房** | 强化版战斗房，多波次、精英怪，奖励更多 |
| **太阳房** | 选元素亲和，获得高品质 Gambit |
| **月亮房** | 二选一强力法术（模仿星座攻击） |
| **回响房** | 获得伙伴（主动或被动） |
| **安德洛墨达酒吧** | 花「星」买属性、法术、Gambit，与星座互动，换血 |
| **鲸鱼洞窟** | 三选一 Gambit 类型 |
| **孔雀遗迹** | 用血量换法术升级 |
| **熔炉** | 全法术+1 级 / 新法术 / 解锁槽位 |
| **和平房** | 回复血/蓝，可花星补治疗次数 |
| **虚空** | 小 Boss，奖励星、Gambit 解锁器 |
| **Boss 房** | 每世界终点 |

### 可借鉴点
1. **难度星级**：房间标星，玩家按当前强度选路线
2. **多类非战斗房**：休息、选奖励、商店、升级、伙伴，不只有战斗
3. **「星」作为路线内货币**：用于酒吧消费、换血，与探索/战斗产出挂钩
4. **星座挑战**：Boss 提供限时/无伤等挑战，完成有额外奖励
5. **世界主题**：每大关有独立主题与机制，增加变化

## 一、节点类型

| 类型 | 英文 ID | 说明 | 出现时机 |
|------|---------|------|----------|
| **战斗** | `combat` | 普通关卡，清敌后选下一路线 | 路线树主节点 |
| **商店** | `shop` | 花灵石买丹药等 | 第一关后作为可选分支（已实现） |
| **事件** | `event` | 奇遇/抉择，奖励或惩罚 | 可选分支，替代一次战斗 |
| **休息** | `rest` | 回复血/蓝，无战斗 | 可选分支 |
| **宝箱** | `chest` | 直接获得物品/灵石，无战斗 | 可选分支 |
| **精英** | `elite` | 强敌，奖励更好 | 可选分支，替代普通战斗 |
| **Boss** | `boss` | 终点 Boss，凯旋条件 | 路线终点 |

## 三、路线结构【已拍板，对标星座战士】

### 整体流程（3 大段 + 段末 Boss + 最终 Boss）

- **3 大段**：每段若干普通关 + **Boss 前必为回复/商店** + **段末 Boss**
- **最后一大段**：段末 Boss₃ 后 → **休息点（回复+商店）** → **最终 Boss** → 凯旋

### 路线结构图

```
大段 1（试炼/秘境/凶地）
  普通关 0,1,2,3,4 → 选路(2/3) → ... → 回复/商店（Boss 前二选一）→ 段末 Boss₁ → 直接进大段 2

大段 2（灵泉/剑冢/丹阁）
  普通关 5,6,7,8,9 → 选路 → ... → 回复/商店 → 段末 Boss₂ → 直接进大段 3

大段 3（灵脉/魔窟/封印塔）
  普通关 10,11,12,13,14 → 选路 → ... → 回复/商店 → 段末 Boss₃ → 休息点（回复+商店）→ 最终 Boss → 凯旋
```

### 关键规则

| 规则 | 说明 |
|------|------|
| **Boss 前** | 进入段末 Boss / 最终 Boss 前的最后一关，只能是回复或商店 |
| **段末 Boss 后** | 无回复/商店，直接进入下一段普通关（显示唯一入口进入） |
| **休息点** | 仅大段 3：段末 Boss₃ 与最终 Boss 之间，同时提供回复+商店 |
| **选路** | 仅发生在普通关（战斗/精英/宝箱/事件）之后 |

### 节点类型汇总【已拍板】

| 类型 | 是否加入 | 说明 |
|------|----------|------|
| 战斗 | ✓ | 普通关 |
| 商店 | ✓ | 花灵石购买 |
| 事件 | ✓ | 奇遇/抉择 |
| 休息 | ✓ | 回复血/蓝 |
| 宝箱 | ✓ | 直接获得奖励 |
| 精英 | ✓ | 强敌，奖励更好 |

| **休息点** | ✓ | 回复+商店（仅大段 3 段末 Boss₃ 与最终 Boss 之间）|

## 四、各类型出现规则

### 战斗（combat）
- 路线树主节点，清敌后进入路线选择
- 每关固定敌人配置（见 levels.py）

### 商店（shop）
- **已实现**：第一关后作为可选分支之一
- **可扩展**：在第二层（4/5/6 后）再出现一次？或作为事件随机触发？

### 事件（event）
- 作为路线选项出现，如「奇遇」「抉择」
- 选事件 = 跳过本层战斗，获得随机奖励/惩罚
- 可与战斗、休息、宝箱并列

### 休息（rest）
- 作为路线选项，回复一定血/蓝
- 不战斗，直接进入下一层路线选择
- 可限制每局出现次数（如最多 1 次）

### 宝箱（chest）
- 作为路线选项，直接获得灵石/丹药
- 不战斗，进入下一层
- 可随机品质（普通/稀有）

### 精英（elite）
- 作为路线选项，替代普通战斗
- 敌人更强，奖励更多（灵石、稀有丹药）
- 可与普通战斗二选一

### Boss（boss）
- **段末 Boss**：每段 1 个，共 3 个，击败后直接进下一段
- **最终 Boss**：仅大段 3，休息点之后，击败即凯旋

## 五、路线选择 UI 扩展【已拍板，对标 Tiny Rogues】

过关后上方显示入口，点击或按 1/2/3 选择。

**2/3 选逻辑**（对标 Tiny Rogues）：
- 关 0 后：2 选
- 关 1～9、5～9、10～14 后（普通关）：3 选
- 关 4、9、14 后（Boss 前最后一关）：2 选（只能选回复或商店）
- 段末 Boss / 休息点 / 回复 / 商店 后：唯一出口时显示单入口，无抽样

**每个选项需显示**：
1. **节点类型**：战斗/商店/休息/宝箱/精英/Boss
2. **奖励倾向**：灵石、饰品、法宝、道韵等，用图标表示（灵石/饰品 出现最多，法宝次之，道韵最小）
3. **无难度星级**：普通/精英/Boss 即涵盖难度，不再单独标星

**关卡名称**：进入关卡时在地图显示，渐变退出；选路界面不显示关卡名。

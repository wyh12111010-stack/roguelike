# 灵根与法宝获取设计

> 已拍板。法宝对齐死亡细胞武器（不分等级），详见 `FABAO_ACCESSORY_DESIGN.md`。

---

## 一、灵根获取

- **全部道韵解锁**：在村子用道韵购买，无其他途径

---

## 二、法宝获取

### 获取方式（二选一或组合）

1. **局内遇到 → 道韵购买**
   - 对局中遇到某法宝后，局外才可用道韵解锁
   - 未遇到过则无法购买

2. **成就解锁 → 可购买**
   - 完成特定成就后，该法宝变为可购买（道韵）

### 解锁门槛（第三层逻辑）

- 部分法宝需**通关 N 次**后才「解锁出现」
- 解锁前：不会出现在对局中，也无法通过上述两种方式获取
- 解锁后：才会在局内遇到或成就中可见，才能用方式 1 或 2 获取

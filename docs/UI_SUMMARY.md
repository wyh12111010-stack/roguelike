## 💡 技术亮点

### 1. 渐变系统
```python
# 线性插值算法
ratio = y / height
r = int(color_top[0] * (1 - ratio) + color_bottom[0] * ratio)
```

### 2. 光晕系统
```python
# 径向渐变 + 加法混合
for i in range(radius, 0, -2):
    alpha = int((i / radius) * intensity * 60)
    pygame.draw.circle(glow_surf, (*color[:3], alpha), center, i)
surface.blit(glow_surf, pos, special_flags=pygame.BLEND_ADD)
```

### 3. 粒子系统
```python
# 物理模拟
self.x += self.vx * dt
self.y += self.vy * dt
self.vy += 5 * dt  # 重力/浮力
alpha = int(255 * (1 - self.age / self.lifetime))  # 渐隐
```

# 星星闪烁

## 📈 性能表现

### 帧率
- 目标：60 FPS
- 实际：55-60 FPS（取决于粒子数量）

### 优化建议
1. 限制粒子数量（MAX_PARTICLES = 100）
2. 降低光晕强度（intensity = 0.3）
3. 减少光晕层数（range(radius, 0, -3)）
4. 缓存常用渐变表面

# 以 RPG 为主的肉鸽：如何做到「持续明显的数值提升」

> 对比构筑型 vs RPG 型，分析主流 RPG 肉鸽的成长设计

---

## 一、两种路线的差异

| | 构筑型（杀戮尖塔、以撒） | RPG 型（小骨、哈迪斯、霓虹序列） |
|---|--------------------------|----------------------------------|
| **核心** | Build 组合、策略选择 | 数值成长、变强感 |
| **爽点** | Build 成型、质变瞬间 | 持续变强、每关都有提升 |
| **成长节奏** | 少而精，每个选择有深度 | 多而频，每关/每层都有成长 |
| **数量** | 卡牌/道具多但槽位或机制限制 | 装备/祝福数量多，获取频繁 |

---

## 二、RPG 肉鸽的典型做法

### 小骨（Skul）
- **6 件装备槽**，满槽时新顶旧
- **每关都有获取**：BOSS 宝箱、收藏家、商店
- **品质分层**：普通→稀有→史诗→传说，高品质更强
- **头骨联动**：装备根据头骨类型给特殊加成，增加搭配深度
- **效果**：一局内可换装多次，持续有「又变强了」的感觉

### 哈迪斯
- **祝福无严格上限**：一局可拿多位神的多个祝福
- **获取频繁**：每层房间选奖励，商店、事件、隐藏房
- **多重成长线**：祝福 + 锤子 + 属性 + 信物
- **稀有度**：普通→稀有→传奇→英雄，升级可强化
- **效果**：几乎每个房间都有成长选择，数值持续爬升

### 霓虹序列（类哈迪斯）
- **四种增益**：词条、特饮、武装、律动链接
- **七种获取途径**：关卡、商店、乱流区、黑市、噪乱之匣等
- **每关可选**：关卡奖励、难度越高奖励越多（难度三可自选类型）
- **效果**：每关都有 2～3 个成长点，持续累积

### 苍翼混沌效应
- **6 个意识结晶槽**
- 但配合**技能、策略、传承**等多线成长
- 单局内还有**潜能、合剂**等即时提升

---

## 三、RPG 肉鸽的共性

1. **获取频率高**：每关/每层都有成长机会，不是隔好几关才拿一次
2. **数量够多**：6 件（小骨）、多祝福（哈迪斯），一局内可累积 10+ 次成长
3. **多重成长线**：不只有饰品，还有祝福/词条/锤子/属性等
4. **品质/稀有度**：用稀有度控制强度，让「开出好的」有惊喜感
5. **先慢后快**：前期成长稳，后期有「量变到质变」的拐点

---

## 四、3 个饰品槽为什么不够

| 问题 | 说明 |
|------|------|
| **获取次数少** | 一局 7 关，若只有 3 槽，最多 3 次成长，平均每 2～3 关才一次 |
| **缺乏持续感** | 中间多关无成长，容易觉得「卡住了」「没变强」 |
| **与 RPG 预期不符** | 玩家期待的是「不断变强」，不是「偶尔换一件」 |

---

## 五、要达到「舒服的 RPG 成长感」可以怎么做

### 方案 A：增加槽位（6～8 个）
- 参考小骨，6 件装备
- 获取机会多，每关都有概率拿
- 满槽后替换，仍有取舍

### 方案 B：饰品 + 其他成长线
- 饰品 3～4 槽，但**不只有饰品**
- 增加：过关属性加成、事件 buff、商店临时强化、伙伴 buff 等
- 多条线叠加，保证每关都有成长

### 方案 C：饰品可升级/强化
- 槽位保持 3～4 个
- 但同一饰品可**多次强化**（过关、灵石、事件）
- 单件成长感强，弥补数量

### 方案 D：混合（推荐）
- **6 个饰品槽**，参考小骨
- **获取频率**：过关二选一、商店、Boss、事件
- **品质分层**：普通/稀有/史诗，控制投放
- 保证每关至少有 1 次成长机会

---

## 六、总结

| 目标 | 建议 |
|------|------|
| **持续明显的数值提升** | 每关都要有成长机会 |
| **RPG 型爽感** | 槽位 6 个左右，或增加其他成长线 |
| **保留取舍** | 满槽替换、品质分层、多线之间的选择 |
| **避免构筑型** | 不要「少而精」，要「多而频」 |

**结论**：3 槽确实偏少，难以支撑 RPG 肉鸽的成长感。建议 6 槽 + 提高获取频率，或 3～4 槽 + 其他成长线补足。

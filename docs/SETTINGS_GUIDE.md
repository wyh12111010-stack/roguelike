# 游戏设置说明（Settings Guide）

本文档说明当前全局设置系统的行为、默认值与实现约束。  
目标：让玩家可理解，让开发可扩展。

## 1. 入口与交互模型

- 打开/关闭设置：`F10`
- 关闭并放弃未应用改动：`ESC`（在设置菜单内）
- 设置采用**暂存模型**：
  - 在菜单中修改时，先写入 pending（暂存）
  - 点击“应用设置”后才写入 `meta` 并持久化
  - 点击“取消改动”会丢弃 pending，恢复到当前已生效配置

## 2. 设置页签

- `显示`
- `音频`
- `游戏`
- `操作`
- `系统`

## 3. 各页功能

### 3.1 显示

- 画布大小（分辨率预设）
- 全屏开关
- 当前窗口尺寸显示

预设分辨率定义于 `config.RESOLUTION_PRESETS`。

### 3.2 音频

- 主音量（Master）
- 背景音乐音量（BGM）
- 音效音量（SFX）

说明：
- BGM 实际音量 = `master_volume * bgm_volume`
- SFX（含 Boss 提示音）实际音量 = `master_volume * sfx_volume`

### 3.3 游戏

- Boss 提示音模式：
  - `关`
  - `标准`
  - `强化`
- 失焦自动暂停（窗口失去焦点时是否自动暂停）
- 退出自动保存（退出时是否自动保存局内进度）

### 3.4 操作

- 可重绑定动作（分页显示）
- 点击动作右侧按键框后，按新键完成重绑
- 冲突处理：若新键已被别的动作使用，执行**自动交换**
- “默认键位”按钮：恢复默认绑定（写入暂存，需应用）

### 3.5 系统

- 应用设置
- 取消改动
- 恢复默认（写入暂存）
- 保存并退出游戏

## 4. 默认值

默认值来源：
- `meta.py`（持久化字段默认值）
- `controls.py`（默认键位）

当前关键默认值：

- `master_volume = 1.0`
- `bgm_volume = 0.8`
- `sfx_volume = 1.0`
- `resolution_index = 0`
- `fullscreen = False`
- `boss_cue_mode = 1`（标准）
- `pause_on_focus_lost = True`
- `autosave_on_exit = True`

## 5. 默认键位（操作）

定义在 `controls.DEFAULT_KEYBINDS`，核心动作如下：

- `move_up`: `W`
- `move_down`: `S`
- `move_left`: `A`
- `move_right`: `D`
- `dash`: `LShift`
- `switch_fabao`: `Q`
- `cast_spell`: `E`
- `partner_skill`: `F`
- `use_potion`: `R`
- `char_panel`: `C`
- `return_village`: `R`

## 6. 保留键与约束

保留键定义在 `controls.RESERVED_KEYS`，当前包括：

- `ESC`
- `F10`

说明：
- 保留键不可用于动作重绑定
- 目的：保证设置菜单始终可打开/关闭，避免“锁死输入”

## 7. 生效时机与持久化

- 点击“应用设置”：
  - 将 pending 写回 `meta`
  - 执行必要实时操作（如分辨率/全屏切换）
  - 调用 `persist_meta()` 落盘
- 点击“取消改动”：
  - 丢弃 pending
  - 不改 `meta`
- 退出逻辑：
  - 若开启 `autosave_on_exit` 且满足可存档条件，尝试保存局内进度

## 8. 扩展建议（后续）

- 增加“键位重置单项”而非全量重置
- 增加手柄按键映射层
- 音频增加 UI/Battle/Ambience 分组
- 显示增加帧率上限、垂直同步、粒子密度

# 游戏设置说明（玩家与开发共用）

本文档说明当前版本的全局设置系统：入口、各项含义、保存机制、以及可配置按键规则。

## 1. 设置入口与基础操作

- 打开/关闭设置：`F10`
- 关闭并放弃改动：`ESC`
- 设置分为 5 个页签：
  - `显示`
  - `音频`
  - `游戏`
  - `操作`
  - `系统`

设置采用“暂存-应用”机制：

- 在设置页里调整参数时，先写入暂存，不立即持久化。
- 点击 `应用设置` 才会提交并写入存档。
- 点击 `取消改动` 或按 `ESC` 会丢弃暂存改动。

## 2. 显示设置

- `画布大小`：切换分辨率预设（见 `config.RESOLUTION_PRESETS`）。
- `全屏`：窗口 / 全屏切换。
- 显示页会展示当前窗口尺寸，便于确认生效结果。

## 3. 音频设置

- `主音量`：总音量系数。
- `BGM 音量`：背景音乐系数。
- `SFX 音量`：音效系数。

当前生效关系：

- BGM 实际音量 = `主音量 * BGM音量`
- SFX（含 Boss 提示音）实际音量 = `主音量 * SFX音量`

## 4. 游戏设置

- `Boss提示音模式`
  - `关`
  - `标准`
  - `强化`
- `失焦自动暂停`
  - 开启时，窗口失去焦点会自动暂停更新流程。
- `退出自动保存`
  - 开启时，退出时若满足保存条件会自动保存局内进度。

## 5. 操作设置（可改键）

支持配置：

- 上移 / 下移 / 左移 / 右移
- 冲刺
- 切换法宝
- 施放法术
- 伙伴技能
- 使用丹药
- 人物面板
- 回村子（死亡/胜利后）

重绑定规则：

- 点击动作右侧键位框后按新键。
- 若新键已被其他动作占用，会自动交换两者键位。
- 保留键不可绑定：`ESC`、`F10`（避免菜单功能失效）。
- 操作页支持分页浏览与“默认键位”重置（写入暂存，需应用后提交）。

## 6. 系统页动作

- `应用设置`：提交暂存改动并持久化。
- `取消改动`：放弃暂存改动。
- `恢复默认`：把暂存改动重置为默认值。
- `保存并退出游戏`：先应用设置，再按自动保存规则退出。

## 7. 持久化字段（MetaData）

设置相关存档字段位于 `meta.MetaData`：

- `master_volume`
- `bgm_volume`
- `sfx_volume`
- `resolution_index`
- `fullscreen`
- `boss_cue_mode`
- `pause_on_focus_lost`
- `autosave_on_exit`
- `keybinds`

缺失字段会回落到默认值，确保旧存档兼容。

## 8. 开发维护建议

- 新增设置项时，保持“暂存 -> 应用”的一致行为。
- 涉及输入系统的配置，优先通过 `controls.py` 统一管理。
- 如新增系统级快捷键，优先加入保留键集合，避免与可改键冲突。

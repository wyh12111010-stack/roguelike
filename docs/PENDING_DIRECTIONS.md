## 三、建议优先级

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 高 | 节点与路线 | 影响关卡流程，与 Boss、商店、事件挂钩 |
| 高 | 数值与平衡 | 影响整体体验，需尽早定基调 |
| 中 | 村民与伙伴 | 已有框架，缺具体名单与能力 |
| 中 | 玩家机制 | 闪避、灵力消耗影响战斗手感 |
| 低 | 事件系统 | 可后续补充 |
| 低 | 剧情与叙事 | 可后续补充 |
| 低 | 大 Boss | 世界观留待 |

## 二、设计亮点

### 1. 成就系统
✅ **多样化目标**：覆盖进度、战斗、挑战、收集、流派等多个维度  
✅ **难度梯度**：从简单（首次通关）到困难（无伤、速通）  
✅ **隐藏成就**：增加探索乐趣  
✅ **解锁奖励**：部分成就解锁特殊法宝（如速度大师解锁离火针）

### 2. 法宝系统
✅ **节奏多样**：极速流（0.25s）到重压流（0.9s），覆盖 3.6 倍攻速差  
✅ **伤害平衡**：低伤高频 vs 高伤低频，DPS 相近但手感不同  
✅ **流派支撑**：近战/远程、快速/慢速、AOE/单体都有对应法宝  
✅ **特殊机制**：反弹、拉扯、环绕等独特玩法

### 3. 饰品系统
✅ **流派定向**：每种流派都有专属饰品支撑  
✅ **元素强化**：五行属性都有对应增强饰品  
✅ **风险收益**：高风险高收益的选择  
✅ **套装联动**：2 件套装产生额外效果，增加构建深度  
✅ **生存/经济**：不同玩家偏好都有选择

## 三、数值设计原则

### 1. 法宝 DPS 平衡
```
极速流：伤害 8-12，攻速 0.25-0.34s
  → DPS = 10 / 0.3 = 33.3

中速流：伤害 15-22，攻速 0.4-0.5s
  → DPS = 18 / 0.45 = 40

重压流：伤害 35-40，攻速 0.85-0.9s
  → DPS = 37.5 / 0.875 = 42.9

结论：DPS 相近（33-43），但手感差异大
```

### 2. 饰品强度平衡
```
基础饰品：伤害+8%，攻速+10%（单维度提升）
流派饰品：伤害+15-20%（条件触发）
风险饰品：伤害+25-35%（带负面或条件）
套装饰品：单件+10%，2 件套+25%（需要配合）

3 个饰品叠加：约 50-80% 战力提升
6 个饰品叠加：约 150-250% 战力提升
```

### 3. 成就难度梯度
```
简单（70%+ 玩家可达成）：
  - 首次通关
  - 击败各 Boss
  - 解锁灵根/法宝

中等（30-50% 玩家可达成）：
  - 通关 5 次
  - 50 连击
  - 单局击杀 100

困难（10-20% 玩家可达成）：
  - 无伤通关
  - 速通 30 分钟
  - 100 连击

极难（<5% 玩家可达成）：
  - 速通 25 分钟
  - 无伤击败 Boss
  - 特殊条件通关
```

## 四、流派支撑分析

### 1. 极速流
**法宝**：鞭、镖、爪、针（4 个）  
**饰品**：疾风佩、疾风节奏、风债符、绝境疾风（4 个）  
**特点**：高频连击，适合触发反应和连击加成

### 2. 重压流
**法宝**：锤、炮、斧、刃（4 个）  
**饰品**：重击冲击、暴烈玉、狂战魂（3 个）  
**特点**：一击制敌，适合打 Boss 和精英

### 3. 范围流
**法宝**：扇、铃、鼓、杖、印、鼎（6 个）  
**饰品**：范围大师、多发、穿透（3 个）  
**特点**：清小怪快，适合高密度战斗

### 4. 单体流
**法宝**：枪、弓、剑、刃（4 个）  
**饰品**：单体专家、暴烈玉、玻璃大炮（3 个）  
**特点**：单发高伤，适合打 Boss

### 5. 元素流
**法宝**：每个属性 3-4 个  
**饰品**：5 个元素强化 + 3 个反应增强 + 3 个五行流派（11 个）  
**特点**：依赖元素反应，伤害爆发高

### 6. 生存流
**法宝**：任意  
**饰品**：护心镜、吸血玉、护盾生成、回春符、重生符（5 个）  
**特点**：容错率高，适合新手

- total_wins：总通关次数
- total_kills：累计击杀数
- total_daoyun：累计道韵
- linggen_unlocked：已解锁灵根数
- fabao_unlocked：已解锁法宝数
- partners_healed：已治疗伙伴数

**Boss 击败标记**：
- boss_1_defeated：妖王
- boss_2_defeated：剑魔
- boss_3_defeated：丹魔
- final_boss_defeated：秽源

**特殊条件**：
- no_damage_boss：无伤击败 Boss

- low_health_boss_kill：低血击败 Boss

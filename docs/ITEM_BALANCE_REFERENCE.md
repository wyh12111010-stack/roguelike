# 肉鸽游戏饰品/装备平衡设计参考

> 小骨、以撒、杀戮尖塔、Hades、死亡细胞、苍翼混沌效应等

---

## 一、槽位与数量限制

| 游戏 | 槽位/限制 | 说明 |
|------|-----------|------|
| **小骨** | 6～9 个装备槽 | 满槽时新装备顶掉旧装备，必须做取舍 |
| **以撒** | 无槽位上限 | 被动道具无限叠加，靠品质与稀有度控制 |
| **杀戮尖塔** | 无上限 | 遗物无限拿，靠稀有度与负面效果平衡 |
| **Hades** | 1 个信物 + 多个祝福 | 信物槽固定，祝福按层获取 |
| **死亡细胞** | 2 武器 + 2 技能 | 强制 4 槽，拾取第三件弹出替换窗口 |
| **苍翼混沌效应** | 6 个意识结晶槽 | 大量结晶可选，6 槽必须取舍 |

**思路**：要么**槽位少**（2～4），要么**无槽位但稀有度/品质分层**。

## 二、死亡细胞（Dead Cells）

### 槽位设计
- **2 武器 + 2 技能**，共 4 槽
- 拾取第三把武器时**弹出窗口**，必须选一把替换
- 双手武器会替换全部两把单手武器
- 护符：提供附加能力 + 卷轴加成（减伤、卷轴数）

### 设计理念
- 曾尝试技能树、背包，最终选择**固定槽位**
- 目的：**强制取舍**，找到新武器时必须决定留哪把
- 每把武器要有**明显不同的机制**，而非单纯数值差异
- 例：双匕首第三击暴击、冰弓冰冻、怨恨之刃受伤后 8 秒暴击

### 平衡手段
- 装备等级（罗马数字）+ 品质（无/+/++/S/L）
- 词缀系统：高等级装备带 1～多个词缀
- 地图等级、掉落位置影响装备强度
- 传奇装备（L）：无伤 BOSS、小概率掉落，伤害取决于双属性之和

## 三、苍翼混沌效应（BlazBlue: Entropy Effect）

### 槽位设计
- **6 个意识结晶槽**，固定上限
- 大量结晶可解锁，但只能戴 6 个，必须取舍

### 结晶类型（按阶段）
- **萌新期**：生机勃勃（+80% HP）、喘息之机（过关+4% HP）、简单粗暴（普攻+20%）
- **进阶期**：嗜血疯狂（连杀+伤害）、技高一筹（技能+20%）、减轻负担（休息清负面）
- **大佬期**：巨人杀手（对 Boss+30%）、命不该绝（复活次数）、合剂强效（回血+80%）

### 平衡手段
- 槽位限制制造取舍
- 按流派搭配：纯伤害、微容错、生存等
- 不同结晶服务不同玩法，避免「万能解」

## 五、品质/稀有度分层

| 游戏 | 分层 | 作用 |
|------|------|------|
| **小骨** | 普通、稀有、史诗、传说 | 高品质更强，获取更难 |
| **以撒** | 品质 0～4 | 不同房间有不同品质池，控制期望 |
| **杀戮尖塔** | 普通、罕见、稀有、Boss | 稀有遗物更强，常带负面 |
| **Hades** | 祝福稀有度 | 稀有祝福效果更强 |

**思路**：用品质控制**强度期望**，稀有 = 强但难拿。

## 六、获取途径与投放

| 游戏 | 做法 |
|------|------|
| **以撒** | 宝箱房、Boss 房、商店、恶魔房、天使房、隐藏房各有**独立道具池**，品质期望不同 |
| **小骨** | BOSS 宝箱、避难所收藏家、商店（金币/暗石英） |
| **杀戮尖塔** | 战斗奖励、商店、事件、Boss，按层数调整遗物池 |

**思路**：
- 不同来源用**不同道具池**
- 稀有来源（Boss、隐藏房）给更高品质
- 商店 = 可控，但价格/刷新做限制

## 七、负面效果与取舍

| 游戏 | 做法 |
|------|------|
| **杀戮尖塔** | Boss 遗物常有负面（如「不能回复」），用负面换强度 |
| **以撒** | 部分道具带负面，或改变玩法（如飞行但减血） |
| **小骨** | 装备多为纯正面，靠槽位限制做取舍 |

**思路**：强饰品可配**可控负面**，或靠**槽位/替换**制造取舍。

## 八、设计原则（来自 RogueLike 神器设计思考）

### 应避免
- 应用场景过窄，只在特定流派/情形有用
- 与核心机制冲突（如目标是不受伤，却奖励受伤）
- 正面效果在某些情况下变成困扰
- 负面过强，压过正面体验
- 纯随机且无法利用，只带来混乱
- 效果太弱，存在感低
- 拿取后无成长甚至负成长
- 过度削弱难度，让高难度失去意义

### 应该注意
- 目的：**强度成长 + 策略选择**
- 引导不同流派，增加重玩价值
- 可做「先抑后扬」（如先亏后赚）
- 可帮玩家解决难点（漏怪、卡组抽干等）
- 可做**联动/套装**：多件配合更强
- 可做**成长型**：随时间/击杀变强
- 可做**任务型**：达成条件后增强

## 十、爽点与取舍的关系

### 爽点从哪来

| 爽点类型 | 说明 | 代表游戏 |
|----------|------|----------|
| **获得感** | 拿到新装备/强装备的瞬间满足 | 以撒开宝箱、小骨开传说 |
| **成长感** | 数值/能力肉眼可见变强 | 死亡细胞词缀、苍翼结晶叠加 |
| **策略验证** | 「我选对了」——Build 成型、碾压敌人 | 杀戮尖塔、死亡细胞 |
| **意外惊喜** | 随机开出稀有，超出预期 | 以撒恶魔房、死亡细胞传奇祭坛 |
| **能力释放** | 用新机制打出连招、清场 | 死亡细胞武器机制、苍翼连杀 |
| **险中求胜** | 艰难抉择后通关的成就感 | 杀戮尖塔 Boss 遗物取舍 |

### 取舍为什么能放大爽点

**1. 稀缺 = 价值感**
- 不能全拿 → 每一个选择都有份量
- 槽位有限 → 拿到好装备是「赚到」，不是理所当然
- 替换时纠结 → 说明每件都有用，选对了更有成就感

**2. 选择 = 认同感**
- 「这是我的 Build」→ 玩家对搭配有归属感
- 取舍让每次选择都算数 → 通关时是「我选出来的胜利」
- 不同取舍 = 不同流派 → 重玩有新鲜感

**3. 取舍 = 发现感**
- 这次选 A，下次选 B → 每次都是新体验
- 没有万能解 → 需要思考、尝试、学习
- 赌对了稀有 → 「我赌对了」比「白送」更爽

**4. 无取舍的后果**
- 全拿 = 什么都强 = 什么都不突出
- 成长曲线平缓 = 缺乏「拐点」式爽感
- 没有策略深度 = 重复感强

### 总结

**爽点 = 成长 + 选择被验证 + 稀缺带来的价值感**

**取舍的作用**：让选择有份量 → 选对了更爽；让获得有稀缺感 → 拿到更爽；让 Build 有认同感 → 通关更爽。

## 十一、对你游戏的建议

1. **槽位**：2～4 个饰品槽，满槽时替换
2. **品质**：普通/稀有/史诗，或简单分 2 档
3. **投放**：商店、事件、Boss、宝箱用不同池
4. **取舍**：强饰品可带小负面，或靠槽位限制
5. **联动**：部分饰品与灵根/法宝有联动效果

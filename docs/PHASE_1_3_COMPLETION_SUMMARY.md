## 🎯 系统设计

### 秽源共鸣系统

```python
# 6 大类侵蚀
狂暴侵蚀：增加敌人伤害
坚韧侵蚀：增加敌人生命
迅捷侵蚀：增加敌人速度
增殖侵蚀：增加敌人数量
混沌侵蚀：敌人随机元素/Boss额外技能
贫瘠侵蚀：减少资源获取

# 3 个等级
轻度：+1 共鸣强度
中度：+2 共鸣强度
极度：+3 共鸣强度

# 同类互斥
只能选择同一类型的一个等级
例如：可以选 轻度狂暴 + 中度坚韧
     不能选 轻度狂暴 + 中度狂暴

# 道韵倍率
每点共鸣强度 +20% 道韵
例如：共鸣强度 5 = +100% 道韵
     共鸣强度 10 = +200% 道韵
```

### 特色饰品设计原则

```
❌ 错误：数值递增
  狂暴碎片：伤害 +8%
  狂暴之核：伤害 +15%
  狂暴之心：伤害 +25%

✅ 正确：横向差异
  狂暴碎片：伤害+12%，生命-10
  狂暴之核：暴击率+15%，攻速-8%
  狂暴之心：连击>10时伤害+30%

→ 每个饰品都有独特机制
→ 没有明显强弱
→ 适合不同流派
```

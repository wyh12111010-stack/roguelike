# 法宝与饰品解锁设计

> 核心问题：
> 1. 道韵价格一直有问题，需要重新设计
> 2. 部分需要成就解锁，成就要与物品本身有联系
> 3. 局外成长如何渐进式呈现

---

## 一、当前问题分析

### 1.1 道韵价格问题

#### 当前设计（unlock.py）
```python
# 灵根解锁
LINGGEN_COST = {
    "fire": 0,      # 免费
    "water": 15,
    "wood": 15,
    "metal": 18,
    "earth": 20,
    "none": 25,
}

# 法宝解锁
FABAO_COST = {
    "sword": 0,     # 免费
    "spell": 15,
    "staff": 18,
    "blade": 20,
    "stone": 22,
    "needle": 24,
    "cauldron": 26,
}

# 饰品解锁
ACCESSORY_UNLOCK_COST = {
    "dmg_s": 12,
    "dmg_pct": 15,
    "atk_spd": 14,
    "pierce": 18,
    "multi": 20,
    ...
}
```

#### 问题
```
1. 价格梯度不明显
   - 灵根：15-25（差距 10）
   - 法宝：15-26（差距 11）
   - 饰品：12-20（差距 8）
   → 差距太小，没有「贵重感」

2. 没有考虑侵蚀度系统
   - 侵蚀度越高，道韵越多
   - 但解锁价格是固定的
   → 后期道韵溢出

3. 没有成就门槛
   - 全部用道韵解锁
   - 缺少「完成挑战解锁」的成就感

4. 没有渐进式呈现
   - 一开始就能看到所有解锁项
   - 缺少「发现新东西」的惊喜
```

## 二、死亡细胞的解锁设计

### 2.1 细胞解锁系统

#### 解锁方式
```
1. 细胞解锁（类似道韵）
   - 用细胞解锁新武器/技能
   - 价格递增：10 → 20 → 50 → 100 → 200 → 500

2. Boss Cell 门槛
   - 某些武器需要 1BC 才能解锁
   - 某些武器需要 2BC 才能解锁
   - 某些武器需要 4BC 才能解锁

3. 成就解锁
   - 某些武器需要完成特定成就
   - 例如：「无伤击败 Boss」解锁特定武器
   - 例如：「用盾反击杀 100 个敌人」解锁盾牌

4. 渐进式呈现
   - 初始只能看到少数解锁项
   - 通关后解锁更多选项
   - 完成成就后解锁隐藏选项
```

#### 关键设计
```
1. 价格梯度明显
   - 基础武器：10-20 细胞
   - 进阶武器：50-100 细胞
   - 高级武器：200-500 细胞
   - 传奇武器：1000+ 细胞

2. 多重门槛
   - 不是「有钱就能买」
   - 需要：细胞 + BC 等级 + 成就

3. 成就与物品关联
   - 盾牌：用盾反击杀敌人
   - 弓：远程击杀敌人
   - 毒武器：用毒击杀敌人
   → 成就与物品玩法相关
```

## 三、我们的解锁设计方案

### 3.1 道韵价格重新设计

#### 设计原则
```
1. 价格梯度明显
   - 基础：10-20 道韵（1-2 局）
   - 进阶：30-50 道韵（3-5 局）
   - 高级：60-100 道韵（6-10 局）
   - 传奇：150-200 道韵（15-20 局）

2. 考虑侵蚀度收益
   - 侵蚀度 0：每局 35 道韵
   - 侵蚀度 1：每局 42 道韵（+20%）
   - 侵蚀度 5：每局 70 道韵（+100%）
   - 侵蚀度 10：每局 105 道韵（+200%）

3. 解锁节奏
   - 前期：快速解锁基础内容（1-2 局）
   - 中期：逐步解锁进阶内容（3-5 局）
   - 后期：长期目标（10+ 局）
```

### 3.3 渐进式呈现设计

#### 设计原则
```
不是一次性显示所有解锁项
而是根据进度逐步显示

阶段 0（初始）：
  - 只显示基础内容
  - 其他内容隐藏（显示「???」）

阶段 1（首次通关）：
  - 显示侵蚀度 1 的解锁项
  - 显示伙伴解锁项（如果治疗了伙伴）

阶段 2（侵蚀度 2）：
  - 显示侵蚀度 2 的解锁项
  - 显示成就解锁项

阶段 3+：
  - 逐步显示更多内容

### 3.4 解锁节奏设计

#### 第 1-2 局（侵蚀度 0）
```
道韵收入：35 × 2 = 70

可解锁：
  - 金灵根（30）
  - 土灵根（30）
  - 基础饰品 2-3 个（15-20 each）

目标：
  - 快速解锁基础内容
  - 让玩家感受到进度
```

#### 第 3-5 局（侵蚀度 1）
```
道韵收入：42 × 3 = 126

可解锁：
  - 无灵根（60）
  - 治疗伙伴（25-45）
  - 伙伴法宝（40-50）
  - 进阶饰品（30-40）

目标：
  - 解锁伙伴系统
  - 解锁进阶内容
```

#### 第 6-10 局（侵蚀度 2-3）
```
道韵收入：50-60 per run

可解锁：
  - 成就法宝（50-60）
  - 侵蚀度法宝（80）
  - 高级饰品（60-80）
  - 局外成长（饰品槽、商店刷新）

目标：
  - 解锁高级内容
  - 解锁局外成长
```

#### 第 10+ 局（侵蚀度 5+）
```
道韵收入：70-100 per run

可解锁：
  - 终极法宝（100-200）
  - 传奇饰品（100+）
  - 终极局外成长

目标：
  - 长期目标
  - 终极挑战
```

## 六、总结

### 核心设计要点

#### 1. 道韵价格梯度
```
基础：10-20（1-2 局）
进阶：30-50（3-5 局）
高级：60-100（6-10 局）
传奇：150-200（15-20 局）
```

#### 2. 多重解锁门槛
```
- 道韵（基础）
- 伙伴 + 道韵（伙伴法宝/饰品）
- 成就 + 道韵（成就法宝/饰品）
- 侵蚀度 + 道韵（高级法宝/饰品）
- 多重条件（终极内容）
```

#### 3. 成就与物品关联
```
- 速度法宝 → 速度成就
- 重击法宝 → 重击成就
- 连击饰品 → 连击成就
- 生存饰品 → 生存成就
```

#### 4. 渐进式呈现
```
- 初始：只显示基础内容

## 🎯 我们的设计方案

### 核心理念
1. **战斗 UI 极简**：只显示必要信息（血、蓝、资源、法宝）
2. **UI 集中一侧**：左上角或左侧
3. **详情按需查看**：按 I 或 Tab 打开角色面板

## 🎨 视觉设计细节

### 左上角 UI 面板

**背景**：
- 半透明黑色 (20, 25, 35, 200)
- 金铜色边框 (180, 150, 90)
- 四角有古风装饰

**血条/蓝条**：
- 血条：红色填充 (220, 60, 60) + 深红背景 (50, 25, 25)
- 蓝条：蓝色填充 (60, 140, 200) + 深蓝背景 (25, 45, 65)
- 高度：20px
- 宽度：140px
- 圆角：2px

**资源显示**：
- 图标：16×16
- 数值：24px 字体
- 灵石：金色 (218, 165, 32)
- 道韵：碧玉色 (64, 224, 208)

**法宝图标**：
- 尺寸：48×48
- 边框：2px
- 当前手持：金色发光边框 + 外发光效果
- 未手持：深色边框
- 名称：12px 字体，米黄色
- 属性：10px 字体，属性颜色

**角色面板按钮**：
- 尺寸：120×30
- 背景：深蓝色 (45, 55, 75)
- 悬停：亮蓝色 (65, 80, 110)
- 文字：18px 字体，米黄色
- 图标：I 键提示

---

### 角色面板

**背景**：
- 半透明黑色覆盖 (0, 0, 0, 180)
- 中央面板：深色背景 (25, 30, 42)
- 金铜色边框 (180, 150, 90)

**布局**：
- 左侧：角色立绘（200×200）
- 右侧：信息面板（550×550）
- 分栏显示：属性、灵根、法宝、饰品

**信息卡片**：
- 背景：(35, 42, 58)
- 边框：(100, 100, 120)
- 圆角：4px
- 间距：10px

**饰品列表**：
- 每个饰品：图标 32×32 + 文字
- 显示等级、效果
- 空槽：灰色虚线框
- 悬停：高亮 + 详细描述

## 🎮 交互设计

### 战斗中

**法宝切换**：
- 按键：Tab 或 Q
- 效果：两个图标高亮切换
- 动画：0.2 秒淡入淡出
- 音效：切换音

**打开角色面板**：
- 按键：I 或 Tab（长按）
- 效果：全屏面板淡入
- 游戏暂停（可选）
- 背景模糊（可选）

---

### 角色面板中

**关闭面板**：
- 按键：ESC 或 I
- 效果：面板淡出

**查看饰品详情**：
- 悬停：显示详细描述
- 点击：无操作（只读）

**滚动**：
- 饰品多时可滚动
- 鼠标滚轮或方向键

## 🚀 总结

### 核心优势
1. **战斗 UI 极简**：只显示必要信息，不干扰战斗
2. **信息集中**：左上角一个面板，视线集中
3. **详情按需**：按 I 打开角色面板，查看完整信息
4. **符合习惯**：参考小骨英雄，玩家熟悉

- 先实现 P0（战斗 UI）
- 再实现 P1（角色面板）

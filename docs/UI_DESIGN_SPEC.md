# UI 设计规格（供 Figma 参考）

> 修仙肉鸽 - 800×600 主分辨率，修真风格

---

## 一、设计系统

### 色彩
| 用途 | 色值 | 说明 |
|------|------|------|
| 血条 | #C8463C | 朱红 |
| 灵力条 | #3C8CC8 | 青灵 |
| 金铜描边 | #B4965A | 条框、按钮边框 |
| 背景深 | #1E1E28 | 遮罩、弹窗底 |
| 主文字 | #C8C8C8 | 常规 UI |
| 道韵 | #B4A064 | 金色调 |
| 灵石 | #96C896 | 青绿 |

### 字体
- 主字体：微软雅黑 / Microsoft YaHei
- 标题：24–28px
- 正文：18–20px
- 小字：16px

### 圆角与间距
- 弹窗圆角：8px
- 按钮圆角：4px
- 条高度：12–14px
- 元素间距：12–16px

---

## 二、战斗 HUD

### 左侧信息区（x: 10）
- 道韵、灵石、丹药：各一行，y 间距 20
- 当前关、剩余敌人：各一行
- [C] 人物 提示
- 伙伴充能条：120×8px，有伙伴时显示

### 底部条（y: 560）
- 血条 / 灵力条并排，各 200×14px，间距 16
- 居中，金铜 1px 描边
- 标签在上方 16px

### 人物页面（C 打开）
- 半透明遮罩 200 alpha
- 标题「人物」居中，y: 50
- 灵根、法宝、饰品：居中，行高 32
- 底部「ESC 或 C 关闭」

## 三、过关结算弹窗 ✅ 已实现

- 尺寸：320×140
- 居中，半透明深底，金铜描边
- 标题：「过关」
- 内容：+N 灵石、+N 道韵（若有）
- 底部提示：点击或 1.5 秒后继续

## 四、成就解锁 Toast ✅ 已实现

- 位置：屏幕顶部居中，y: 80
- 尺寸：自适应宽度×40
- 背景：半透明深色，金铜描边
- 文案：「解锁成就：xxx」
- 显示时长：2 秒，最后 0.5 秒淡出

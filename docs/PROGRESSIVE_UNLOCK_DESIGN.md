# 渐进式解锁与难度递增系统设计

> 基于 Hades 热度系统 + 星座战士觉醒系统
> 核心：不一次性抛给玩家，而是逐步解锁

---

## 一、Hades 和星座战士的设计智慧

### 1.1 Hades 的核心机制

#### 热度系统（Heat System）
```
通关后解锁「契约」系统：
  - 每个契约增加难度，获得热度值
  - 热度越高，奖励越好
  - 玩家自己选择难度

契约示例：
  - 敌人伤害+20%（+1 热度）
  - 敌人生命+30%（+1 热度）
  - 敌人移速+20%（+1 热度）
  - Boss 额外技能（+2 热度）
  - 陷阱伤害+100%（+1 热度）
  - 时间限制（+2 热度）

关键：
  - 玩家自己控制难度
  - 高难度 = 高奖励
  - 可以逐步提升
```

#### 渐进式解锁
```
第 1 次通关：解锁热度系统
第 2 次通关：解锁新武器
第 3 次通关：解锁新房间
第 5 次通关：解锁新 NPC
第 10 次通关：解锁真结局

局外成长：
  - 镜子：用黑暗（货币）升级属性
  - 承包商：用宝石解锁新房间/装饰
  - 武器：用泰坦之血升级
  - 伙伴：用花蜜提升好感度

关键：
  - 不是一次性全给
  - 每次通关都有新东西解锁
  - 持续的新鲜感
```

### 1.2 星座战士的核心机制

#### 觉醒系统（Awakening）
```
每次通关后，世界「觉醒」一次：
  - 敌人变强
  - 新敌人类型出现
  - 新 Boss 技能解锁
  - 新事件出现

觉醒等级：
  觉醒 0：基础难度
  觉醒 1：敌人+20% 伤害，新敌人类型
  觉醒 2：敌人+40% 伤害，Boss 新技能
  觉醒 3：敌人+60% 伤害，新精英类型
  ...
  觉醒 10：终极挑战

关键：
  - 自动递增，不需要手动选择
  - 每次觉醒都有新内容
  - 难度逐步提升
```

#### 渐进式解锁
```
觉醒 0：基础内容
觉醒 1：解锁新法术
觉醒 2：解锁新饰品
觉醒 3：解锁新角色
觉醒 5：解锁新模式
觉醒 10：解锁真结局

局外成长：
  - 法术树：用灵魂升级
  - 饰品池：逐步解锁
  - 角色：完成挑战解锁
  - 成就：解锁永久加成

关键：
  - 每次觉醒都有新东西
  - 不是一次性全给
  - 持续的目标感
```

## 二、修仙肉鸽的设计方案

### 2.1 核心概念：「天劫」系统

#### 世界观融合
```
设定：
  每次击败秽源，世界净化一次
  但秽源不会真正死亡，会在更深处重生
  下次出现时，秽源会更强
  这就是「天劫」——修真者必须面对的考验

天劫等级：
  天劫 0：初次闯关（基础难度）
  天劫 1：一劫（秽源重生，世界变化）
  天劫 2：二劫（秽源更强，新威胁出现）
  ...
  天劫 9：九劫（终极挑战，飞升成仙）
```

### 2.2 天劫系统设计

#### 自动递增 + 手动调节
```
基础机制：
  - 每次通关后，天劫等级自动+1
  - 天劫等级影响难度和奖励
  - 玩家可以选择「降劫」（降低难度，但奖励减少）

天劫等级效果：
  天劫 0：基础难度
  天劫 1：敌人伤害+15%，道韵+20%
  天劫 2：敌人伤害+30%，生命+20%，道韵+40%
  天劫 3：敌人伤害+45%，生命+40%，新敌人类型，道韵+60%
  天劫 4：敌人伤害+60%，生命+60%，精英出现率+30%，道韵+80%
  天劫 5：敌人伤害+75%，生命+80%，Boss 新技能，道韵+100%
  ...
  天劫 9：敌人伤害+150%，生命+180%，终极挑战，道韵+200%

降劫机制：
  - 在村子中可以选择「降劫」
  - 降低 1 级天劫，道韵奖励-30%
  - 最低降到天劫 0
  - 用于：卡关时降低难度，或测试新 Build
```

#### 天劫带来的变化
```
天劫 1：
  - 解锁「劫难印记」系统（类似 Hades 契约）
  - 新敌人类型：「劫兽」（被秽源侵蚀的妖兽）

天劫 2：
  - 解锁新法宝：「渡劫剑」
  - 新事件：「劫云」（危险但高奖励）

天劫 3：
  - 解锁新灵根：「劫雷灵根」
  - Boss 获得新技能

天劫 4：
  - 解锁新饰品池
  - 精英敌人出现率提升

天劫 5：
  - 解锁新伙伴：「渡劫者」
  - 新节点类型：「劫难试炼」

天劫 6-8：
  - 持续解锁新内容
  - 难度持续提升

天劫 9：
  - 终极挑战
  - 真结局：「飞升成仙」
```

### 2.3 劫难印记系统（可选难度调节）

#### 类似 Hades 契约，但修仙化
```
在村子中可以激活「劫难印记」：
  - 每个印记增加难度，获得额外道韵
  - 玩家自己选择激活哪些
  - 可以叠加多个印记

印记列表：

【秽源之怒】
  - 敌人伤害+20%
  - 额外道韵+15%

【秽源之力】
  - 敌人生命+30%
  - 额外道韵+15%

【秽源之速】
  - 敌人移速+25%
  - 额外道韵+10%

【秽源之众】
  - 敌人数量+30%
  - 额外道韵+20%

【秽源之智】
  - 敌人攻击频率+30%
  - 额外道韵+15%

【劫雷降世】
  - 战斗中随机降下劫雷（高伤害）
  - 额外道韵+25%

【灵石匮乏】
  - 灵石掉落-30%
  - 额外道韵+20%

【丹药稀缺】
  - 丹药上限-1
  - 额外道韵+15%

【时间劫难】
  - 每关有时间限制（3 分钟）
  - 额外道韵+30%

【Boss 强化】
  - Boss 获得额外技能
  - 额外道韵+35%

【极限挑战】
  - 玩家生命上限-20%
  - 额外道韵+40%

【劫难轮回】
  - 死亡后从第 1 关重新开始（不是从村子）
  - 额外道韵+50%

关键：
  - 玩家自己控制难度
  - 高风险高收益
  - 可以逐步尝试
```

### 2.4 渐进式局外成长解锁

#### 当前问题
```
现在的设计：
  - 一开始就能看到所有局外成长选项
  - 玩家不知道该先升级什么
  - 缺少「发现新东西」的惊喜

问题：
  - 选择过载
  - 缺少目标感
  - 缺少新鲜感
```

#### 新设计：分阶段解锁

##### 阶段 0：初始状态（第 1 局）
```
可见内容：
  - 5 位伙伴（但都未治疗）
  - 基础灵根（火/水/木）
  - 基础法宝（剑/符/杖）
  - 中央广场（栖霞）

不可见内容：
  - 其他灵根
  - 其他法宝
  - 饰品槽扩展
  - 商店刷新
  - 开局饰品
  - 生命/灵力成长
  - 丹药上限

提示：
  「击败秽源后，栖霞村将发生变化...」
```

##### 阶段 1：首次通关后
```
解锁：
  - 天劫系统（显示天劫等级）
  - 玄真的灵根殿（可以解锁新灵根）
  - 铸心的炼器坊（可以解锁新法宝）
  - 新灵根：金/土（需要道韵解锁）
  - 新法宝：刃/印（需要道韵解锁）

提示：
  「栖霞村的封印松动了，新的力量开始显现...」
  「玄真：你已经证明了自己，我可以教你更多灵根之法」
  「铸心：你的实力足以驾驭更强的法宝了」
```

##### 阶段 2：第 2 次通关后（天劫 1）
```
解锁：
  - 碧落的丹阁（生命/灵力/丹药上限成长）
  - 饰品槽扩展选项（6→7）
  - 新灵根：无灵根
  - 新法宝：针/鼎

提示：
  「碧落：你的身体已经能承受更强的力量了」
  「铸心：我可以为你扩展饰品槽位了」
```

##### 阶段 3：第 3 次通关后（天劫 2）
```
解锁：
  - 商店刷新选项（1→2）
  - 劫难印记系统（可选难度调节）
  - 新法宝池（通过伙伴解锁）

提示：
  「铸心：我研究出了刷新商品的方法」
  「栖霞：你可以尝试劫难印记，挑战更高难度」
```

##### 阶段 4：第 5 次通关后（天劫 4）
```
解锁：
  - 开局饰品选项（随机 1 个）
  - 饰品槽扩展（7→8）
  - 商店刷新（2→3）
  - 新饰品池

提示：
  「铸心：你可以在出发前携带一件饰品了」
```

##### 阶段 5：第 10 次通关后（天劫 9）
```
解锁：
  - 真结局：「飞升成仙」
  - 饰品槽扩展（8→9）
  - 开局饰品（二选一）
  - 隐藏内容

提示：
  「栖霞：你已经接近飞升的境界了...」
```

### 2.5 伙伴解锁的渐进式设计

#### 当前问题
```
现在：
  - 5 位伙伴一开始就都在
  - 只是需要道韵治疗

问题：
  - 缺少「发现新伙伴」的惊喜
```

#### 新设计：逐步出现

##### 初始状态
```
可见伙伴：
  - 玄霄（雷宗掌门）- 已在村中
  - 青璃（青丘狐族长老）- 已在村中

不可见伙伴：
  - 赤渊（散修剑尊）- 未出现
  - 碧落（丹道宗师）- 未出现
  - 墨羽（影宗宗主）- 未出现
```

##### 解锁条件
```
赤渊：
  - 首次通关后出现
  - 剧情：「听闻你击败了秽源，特来拜访」

碧落：
  - 第 2 次通关后出现
  - 剧情：「你的伤势引起了我的注意」

墨羽：
  - 第 5 次通关后出现
  - 剧情：「你已经有资格知道真相了...」
  - 带来隐藏剧情
```

### 2.6 成就系统的渐进式解锁

#### 当前问题
```
现在：
  - 所有成就一开始就可见
  - 缺少「发现隐藏成就」的惊喜
```

#### 新设计：分类 + 隐藏

##### 成就分类
```
【基础成就】（一开始可见）
  - 首次通关
  - 击败各 Boss
  - 解锁灵根/法宝
  - 累计击杀数

【进阶成就】（通关 1 次后解锁）
  - 无伤通关某关
  - 特定流派通关
  - 速通挑战

【隐藏成就】（触发后才显示）
  - 特殊事件
  - 特殊组合
  - 彩蛋

【天劫成就】（天劫 1 后解锁）
  - 天劫 N 通关
  - 激活 N 个劫难印记通关
  - 极限挑战
```

## 六、关键要点总结

### 核心设计思路
```
1. 天劫系统：自动递增难度 + 自动解锁内容
2. 劫难印记：手动调节难度 + 额外奖励
3. 渐进式解锁：不一次性抛给玩家
4. 持续目标：每次通关都有新东西
```

### 与 Hades/星座战士的对比
```
Hades：
  - 热度系统（手动调节）
  - 渐进式解锁
  - 我们借鉴：劫难印记 + 渐进式解锁

星座战士：
  - 觉醒系统（自动递增）
  - 每次觉醒新内容
  - 我们借鉴：天劫系统 + 内容解锁

修仙肉鸽：
  - 天劫系统（自动递增）+ 劫难印记（手动调节）
  - 渐进式解锁局外成长
  - 修仙世界观融合
```

### 玩家体验
```
第 1 局：
  「这游戏还不错」

第 1 次通关：
  「哇，解锁了新东西！」
  「天劫系统是什么？」
  「新伙伴出现了！」

第 2-5 次通关：
  「每次都有新东西解锁」
  「难度在提升，但我也在变强」
  「劫难印记可以挑战更高难度」

第 10 次通关：
  「终于到天劫 9 了」
  「真结局！」
  「飞升成仙！」
```

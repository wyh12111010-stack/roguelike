# 解锁系统设计（修正版）

> 核心修正：
> 1. 侵蚀度是玩家自主选择的（不是自动递增）
> 2. 法宝之间是平衡的，没有「终极法宝」
> 3. 差别只在攻击方式，不在强度

---

## 一、侵蚀度系统（修正）

### 1.1 核心机制

```
侵蚀度是玩家自主选择的难度：
  - 通关后解锁更高侵蚀度
  - 但玩家可以自由选择任意已解锁的侵蚀度
  - 不是「必须打侵蚀度 N 才能打 N+1」

例如：
  第 1 局：侵蚀度 0（唯一选择）
  第 2 局：可选侵蚀度 0 或 1（通关后解锁了 1）
  第 3 局：可选侵蚀度 0、1 或 2（如果通关了 1）
  
  玩家可以：
    - 一直刷侵蚀度 0（简单，道韵少）
    - 挑战侵蚀度 5（困难，道韵多）
    - 回去刷侵蚀度 2（为了特定掉落）
```

### 1.2 解锁条件

```
侵蚀度 0：初始可用
侵蚀度 1：通关侵蚀度 0 后解锁
侵蚀度 2：通关侵蚀度 1 后解锁
侵蚀度 3：通关侵蚀度 2 后解锁
...
侵蚀度 10：通关侵蚀度 9 后解锁

关键：
  - 必须通关侵蚀度 N 才能解锁 N+1
  - 但解锁后可以自由选择
  - 可以反复刷任意侵蚀度
```

## 二、法宝解锁设计（修正）

### 2.1 核心原则

```
法宝之间是平衡的：
  - 没有「基础法宝」vs「终极法宝」
  - 只有攻击方式差异
  - 近战 vs 远程
  - 快速 vs 慢速
  - 弧线 vs 穿透 vs 扇形 vs 重击

解锁门槛的意义：
  - 不是「越贵越强」
  - 而是「越特殊越难解锁」
  - 基础攻击方式：容易解锁
  - 特殊机制：需要成就/侵蚀度
```

### 2.2 法宝分类与解锁

#### 基础法宝（初始免费）
```python
# 初始就有，覆盖基本攻击方式
"sword": 0,     # 赤炎剑 - 近战弧线
"spell": 0,     # 玄水符 - 远程穿透
"staff": 0,     # 青木杖 - 远程扇形
```

#### 伙伴法宝（治疗伙伴后解锁）
```python
# 需要治疗对应伙伴 + 道韵
"blade": ("partner", "chiyuan", 40),    # 庚金刃 - 近战重击
"stone": ("partner", "biluo", 40),      # 厚土印 - 远程抛物
"cauldron": ("partner", "moyu", 50),    # 玄坤鼎 - 远程抛物重击

# 攻击方式差异，不是强度差异
```

#### 成就法宝（完成成就后解锁）
```python
# 需要完成相关成就 + 道韵
"needle": ("achievement", "speed_master", 50),
  # 离火针 - 远程穿透极速
  # 成就：单局攻速达到 200%
  # 攻击方式：与玄水符类似，但更快

"whip": ("achievement", "combo_king", 60),
  # 寒冰鞭 - 近战弧线极速
  # 成就：连续攻击同一目标 20 次
  # 攻击方式：与赤炎剑类似，但更快

"hammer": ("achievement", "heavy_hitter", 50),
  # 雷霆锤 - 近战重击范围
  # 成就：单次攻击造成 100+ 伤害
  # 攻击方式：慢速高伤 AOE

"dart": ("achievement", "sniper", 60),
  # 暗影镖 - 远程穿透极速
  # 成就：从 300 距离外击杀敌人 50 次
  # 攻击方式：远程快速单体

# 关键：成就与法宝玩法相关
```

#### 侵蚀度法宝（达到侵蚀度后解锁）
```python
# 需要达到特定侵蚀度 + 道韵
"mirror": ("erosion", 3, 80),
  # 玄冰镜 - 远程反弹
  # 侵蚀度 3 后可见
  # 攻击方式：特殊机制（反弹）

"chain": ("erosion", 5, 100),
  # 锁魂链 - 近战链条
  # 侵蚀度 5 后可见
  # 攻击方式：特殊机制（拉扯）

"fan": ("erosion", 2, 70),
  # 烈焰扇 - 远程扇形范围
  # 侵蚀度 2 后可见
  # 攻击方式：大范围扇形

# 关键：特殊机制需要更高侵蚀度
```

### 2.3 法宝数值平衡

```
所有法宝 DPS 相近：
  赤炎剑：25/0.4 = 62.5 DPS
  玄水符：15/0.6 = 25 DPS（但远程安全）
  青木杖：18/0.5 = 36 DPS（扇形多发）
  庚金刃：22/0.55 = 40 DPS（重击）
  离火针：12/0.34 = 35.3 DPS（极速）
  雷霆锤：38/0.85 = 44.7 DPS（慢速高伤）

关键：
  - DPS 相近（25-65）
  - 差别在攻击方式
  - 没有「更强」，只有「不同」
```

## 三、道韵价格设计（修正）

### 3.1 定价原则

```
不是「越贵越强」
而是「越特殊/越难获得越贵」

基础法宝（0 道韵）：
  - 初始就有
  - 覆盖基本攻击方式

伙伴法宝（40-50 道韵）：
  - 需要治疗伙伴
  - 1-2 局可解锁

成就法宝（50-60 道韵）：
  - 需要完成成就
  - 2-3 局可解锁（如果完成成就）

侵蚀度法宝（70-100 道韵）：
  - 需要达到侵蚀度
  - 特殊机制
  - 3-5 局可解锁
```

## 六、解锁节奏设计

### 6.1 道韵收入

```
侵蚀度 0：35 道韵/局
侵蚀度 1：42 道韵/局（+20%）
侵蚀度 2：49 道韵/局（+40%）
侵蚀度 3：56 道韵/局（+60%）
侵蚀度 5：70 道韵/局（+100%）
侵蚀度 10：105 道韵/局（+200%）
```

### 6.2 解锁时间线

#### 第 1-2 局（侵蚀度 0）
```
道韵：35 × 2 = 70

可解锁：
  - 金/土灵根（30 each）
  - 基础饰品 2-3 个

目标：快速解锁基础内容
```

#### 第 3-5 局（侵蚀度 1）
```
道韵：42 × 3 = 126

可解锁：
  - 治疗赤渊（25）
  - 庚金刃（40）
  - 无灵根（60）
  - 进阶饰品

目标：解锁伙伴法宝
```

#### 第 6-10 局（侵蚀度 2-3）
```
道韵：50-60 per run

可解锁：
  - 成就法宝（50-60）
  - 侵蚀度法宝（70-80）
```

## 七、总结

### 核心设计要点

#### 1. 侵蚀度系统
```
- 玩家自主选择难度
- 通关后解锁更高侵蚀度
- 可以反复刷任意侵蚀度
- 每个侵蚀度都有独特掉落
```

#### 2. 法宝平衡
```
- 所有法宝 DPS 相近
- 差别只在攻击方式
- 没有「更强」，只有「不同」
- 解锁门槛不代表强度
```

#### 3. 多重解锁门槛
```
- 基础：道韵
- 伙伴：治疗伙伴 + 道韵
- 成就：完成成就 + 道韵
- 侵蚀度：达到侵蚀度 + 道韵
```

#### 4. 渐进式呈现
```
- 初始：只显示基础内容
- 通关后：逐步解锁新选项
- 完成挑战：解锁隐藏内容
- 持续惊喜：每次都有新发现
```

## 系统集成点

### 玩家初始化流程
```
game._start_combat()
  ↓
创建 Player 对象
  ↓
应用基础属性加成（meta.base_health_bonus, meta.base_mana_bonus）
  ↓
player.apply_meta_bonuses() - 应用局外永久加成
  ↓
player.apply_erosion_effects() - 应用侵蚀度效果（如果已解锁）
```

### 通关流程
```
game._show_route_selection()
  ↓
检测通关（sampled 为空）
  ↓
收集局内统计数据（max_attack_speed, max_combo, total_damage）
  ↓
meta.on_win(daoyun, stats)
  ↓
  ├─ 增加通关次数
  ├─ 增加道韵
  ├─ 检查成就（check_achievements）
  └─ 解锁功能（check_unlock_features）
  ↓
显示新解锁的成就
```

### 村庄显示流程
```
VillageScene.draw_village_scene()
  ↓
获取 unlocked_features
  ↓
根据解锁状态控制显示：
  ├─ accessory_unlock: 显示饰品解锁
  ├─ meta_growth: 显示局外成长选项
  ├─ erosion_system: 显示侵蚀度系统
  └─ calamity_seal: 显示劫难印记（未来）
```

## 游戏简介

### 世界观

- **栖霞村**：修真界最后的净土，村中皆是顶尖大能
- **秽源**：纯粹的恶意，侵染着整个世界
- **你**：拥有轮回之力，唯一能自由出入结界的人

### 游戏类型

俯视角动作 Roguelike，每次闯关都是全新体验。

## 操作说明

### 基础操作

| 按键 | 功能 |
|------|------|
| **WASD** | 移动（需要英文输入法） |
| **鼠标移动** | 瞄准方向 |
| **鼠标左键** | 攻击 |
| **空格** | 冲刺/闪避 |
| **E** | 使用法宝技能 |
| **R** | 使用丹药（回血） |
| **1/2/3** | 切换法宝（如果有多个） |

### 系统操作

| 按键 | 功能 |
|------|------|
| **F10** | 打开/关闭设置 |
| **ESC** | 取消/返回 |

### 重要提示

⚠️ **如果 WASD 无反应，请切换到英文输入法！**

## 游戏流程

### 1. 村子准备

游戏开始时，你在栖霞村中：

1. **选择灵根**：决定你的属性（火/水/木/金/土/无）
2. **选择法宝**：决定你的攻击方式（剑/符/杖/刃/印等）
3. **拜访伙伴**：治疗伤势、获得羁绊加成
4. **解锁成长**：用道韵解锁新灵根、法宝、饰品槽等
5. **点击「外出」**：开始闯关

### 2. 战斗关卡

进入关卡后：

1. **消灭所有敌人**：用 WASD 移动，鼠标瞄准，左键攻击
2. **注意走位**：敌人会追击、射击、冲锋，要灵活躲避
3. **使用技能**：按 E 释放法宝技能（消耗灵力）
4. **及时回血**：按 R 使用丹药（数量有限）

### 3. 路线选择

清空敌人后，上方出现传送门，选择下一个节点：

- **战斗**：继续战斗，获得奖励
- **商店**：购买法宝、饰品、道韵碎片
- **休息**：回复生命和灵力
- **宝箱**：获得随机奖励
- **事件**：触发随机事件，可能有风险和收益
- **精英**：更强的敌人，更好的奖励
- **Boss**：章节 Boss，击败后进入下一章节

### 4. 结局

- **凯旋**：击败最终 Boss，获得大量道韵
- **身陨**：生命归零，本局结束，不获得道韵

无论凯旋还是身陨，都会回到村子，保留局外成长。

## 核心系统

### 灵根系统

灵根决定你的属性，影响攻击效果：

| 灵根 | 属性 | 特点 |
|------|------|------|
| **火灵根** | 火 | 高伤害，克制金属性 |
| **水灵根** | 水 | 控制，克制火属性 |
| **木灵根** | 木 | 持续伤害，克制土属性 |
| **金灵根** | 金 | 暴击，克制木属性 |
| **土灵根** | 土 | 防御，克制水属性 |
| **无灵根** | 无 | 平衡，无克制关系 |

### 属性相合与反应

- **相合**：灵根属性 = 法宝属性，伤害 +20%
- **相克**：触发元素反应，产生额外效果
  - 火克金：融化，回复生命
  - 水克火：蒸发，范围溅射
  - 木克土：感电，传导伤害
  - 金克木：斩木，额外伤害
  - 土克水：堵水，减速

### 法宝系统

法宝决定攻击方式：

- **近战法宝**：剑、刃、锤等，近距离高伤害
- **远程法宝**：符、杖、针等，远距离攻击
- **特殊法宝**：扇、镜、链等，独特机制

每个法宝有独特的攻击形态：
- **弧线挥砍**：宽弧近战
- **直线穿透**：单发穿透弹
- **扇形三连发**：一次 3 发
- **蓄力重斩**：慢速高伤
- **抛物线 AOE**：抛物落地爆炸

### 饰品系统

饰品在局内获得，提供被动效果：

- **基础属性**：伤害、攻速、生命、灵力
- **元素增强**：增强特定属性或反应
- **流派定向**：增强近战/远程/快速/重击
- **风险收益**：高收益配高风险

饰品可以升级，变得更强。

### 局外成长

用道韵在村子中解锁永久成长：

| 成长项 | 效果 | NPC |
|--------|------|-----|
| **灵根解锁** | 扩大灵根选择池 | 玄真 |
| **法宝解锁** | 扩大法宝选择池 | 铸心 |
| **饰品槽位** | 增加饰品槽（6→9） | 铸心 |
| **商店刷新** | 增加刷新次数（1→3） | 铸心 |
| **开局饰品** | 开局携带饰品 | 铸心 |
| **生命/灵力** | 提升上限 | 碧落 |
| **丹药上限** | 增加丹药存量 | 碧落 |
| **伙伴治疗** | 解锁伙伴能力 | 各伙伴 |

## 进阶技巧

### 战斗技巧

1. **走位优先**：不要贪刀，活着才有输出
2. **利用地形**：敌人会被边界阻挡
3. **优先目标**：先打远程和召唤型敌人
4. **技能时机**：法宝技能有冷却，关键时刻使用
5. **属性克制**：利用元素反应打出额外伤害

### 构筑技巧

1. **流派一致**：选择相互配合的灵根、法宝、饰品
2. **平衡发展**：不要只堆伤害，生存也很重要
3. **经济管理**：灵石有限，优先买核心物品
4. **商店刷新**：看到好东西可以刷新商店
5. **风险评估**：精英和 Boss 奖励好，但也更危险

### 局外成长策略

1. **优先解锁**：先解锁更多灵根和法宝，增加选择
2. **饰品槽位**：多一个槽位 = 多一个能力
3. **商店刷新**：刷新次数很重要，能找到更好的物品
4. **伙伴治疗**：伙伴能力很强，值得投资

## 常见问题

### Q: WASD 无法移动？
**A:** 请切换到英文输入法。中文输入法会拦截按键。

### Q: 如何保存游戏？
**A:** 游戏自动保存。退出时会保存局外进度，战斗中退出会保存局内进度。

### Q: 死亡后会失去什么？
**A:** 本局的饰品、灵石会清空，但道韵和局外成长保留。

### Q: 如何获得道韵？
**A:** 击败精英（+5）、Boss（+8/12/16）、凯旋（+35）、战斗奖励（小概率 +3）。

### Q: 商店什么时候出现？
**A:** 第一关后就会在路线选择中出现商店节点。

### Q: 如何切换法宝？
**A:** 按 1/2/3 键切换。需要先在商店购买或通关奖励获得。

### Q: 饰品槽满了怎么办？
**A:** 可以在村子用道韵扩展槽位（最多 9 个）。

### Q: 如何查看成就？
**A:** 在村子中央广场，点击「与栖霞对话」或「成就」。

### Q: 游戏卡顿怎么办？
**A:** 按 F10 打开设置，降低分辨率或关闭全屏。

### Q: 如何自定义按键？
**A:** 按 F10 打开设置，切换到「操作」标签页，点击按键框后按新键。

## 二、必须实现（可行游戏）

### 1. 村子（准备阶段）
- **作用**：出生地、战前准备
- **内容**：选灵根、选法宝、道韵/解锁、外出出口
- **流程**：在村里完成选择，从出口进入战斗阶段

### 2. 外出（战斗阶段）
- **作用**：被污染的外界，闯关战斗
- **内容**：关卡、路线选择、敌人
- **流程**：身陨→回村子；凯旋→回村子

### 3. 外出出口
- **作用**：从准备阶段进入战斗阶段
- **内容**：村子内出口，点击后进入战斗

### 4. 回村/轮回
- **作用**：身陨/凯旋后回到准备阶段
- **内容**：战斗结束→回村子→可再次准备、外出

### 5. 道韵消耗（解锁界面）
- **作用**：道韵有地方花，局外成长可见
- **内容**：村子内可打开「解锁」界面，花道韵解锁灵根/法宝

---

## 三、实现顺序

```
1. 村子主界面（出生地）
   └── 选灵根、选法宝、道韵、解锁、外出出口

2. 流程改造
   └── 开局进村子，在村里选灵根+法宝，点出口外出→战斗
   └── 身陨/凯旋→回村子（可再次选、外出）

3. 道韵消耗
   └── 村子内解锁界面，花道韵解锁灵根/法宝
```

## 五、最小可行流程

```
村子（准备阶段）
  ├── 选灵根
  ├── 选法宝
  ├── 查看道韵、解锁灵根/法宝（花道韵）
  └── 外出出口
        │
        ▼
  外出（战斗阶段）
  ├── 关卡 + 路线选择
  ├── 身陨 → 结算道韵 → 回村子（准备阶段）
  └── 凯旋 → 结算道韵 → 回村子（准备阶段）
```
